# **广东省考政治理论体系完整精华笔记**

## **第一部分：马克思主义基本原理**

本部分是政治理论的基石，涵盖哲学、政治经济学和科学社会主义三大板块。

### **一、马克思主义哲学**

#### **1\. 哲学基本问题与派别**

哲学的基本问题是**思维和存在的关系问题**（即意识和物质的关系问题）。

| 派别 | 核心观点 | 历史形态/细分 | 备注/考点 |
| :---- | :---- | :---- | :---- |
| **唯物主义** | **物质第一性**（物质决定意识） | **古代朴素唯物主义**：将物质归结为具体的**实物**（金、木、水、火、土、气）。 **近代形而上学唯物主义**：将物质归结为**原子**；缺陷是孤立、静止、片面。 **辩证唯物主义与历史唯物主义**：认为物质的唯一特性是**客观实在性**。 | 马克思主义属于第三种。 |
| **唯心主义** | **意识第一性**（意识决定物质） | **主观唯心主义**：人的精神（感觉、心、意志）是本原（如：心外无物、存在即被感知）。 **客观唯心主义**：脱离人的“客观精神”（理、道、上帝）是本原（如：理在事先）。 |  |
| **可知论** | 思维能正确反映存在 | 马克思主义坚持可知论。 | 回答了“有无同一性”问题。 |
| **不可知论** | 思维不能或不能完全反映存在 | 否认认识世界的可能性。 |  |

#### **2\. 唯物论（物质与意识）**

*   
  **物质：** 标志客观实在的哲学范畴。**客观实在性**是物质的唯一特性 。  
*   
  **运动：** 物质的根本属性和存在方式。**静止**是运动的特殊状态。世界是绝对运动和相对静止的统一 。  
* **意识：**  
  *   
    **本质：** 人脑对客观存在的主观映象（形式主观，内容客观） 。  
  *   
    **辩证关系：** 物质决定意识；意识具有**主观能动性**（自觉能动性），对物质具有能动的反作用。正确的意识促进事物发展，错误的意识阻碍发展 。  
  *   
    **前提：** **尊重客观规律**是正确发挥主观能动性的前提 。  
  *   
    **人工智能：** 是人意识的延伸，是机械物理过程，**无自觉能动性**，不能完全取代人类意识 。

#### **3\. 唯物辩证法（两大特征：联系与发展）**

* **普遍联系：**  
  *   
    **特征：** 客观性（不以意志为转移）、普遍性（万事万物皆有联系）、多样性、条件性 。  
  *   
    **整体与部分：** 整体居主导地位，统率部分；部分影响整体，**关键部分**甚至对整体功能起决定作用（如“木桶原理”、“牵一发而动全身”） 。  
  *   
    **系统观念：** 体现了普遍联系原理 。  
* **永恒发展：**  
  *   
    **实质：** **新事物的产生和旧事物的灭亡** 。  
  *   
    **判断标准：** 是否符合事物发展的必然趋势（而非时间先后或力量强弱） 。

#### **4\. 唯物辩证法的三大规律**

| 规律名称 | 地位/作用 | 核心原理 | 方法论/启示 |
| :---- | :---- | :---- | :---- |
| **对立统一规律** |  **实质和核心**；揭示发展的**源泉和动力** | **矛盾属性**：同一性（相互依存、贯通/转化）；斗争性（相互排斥、分离）。 **普遍性与特殊性**：普遍性（共性、无条件）；特殊性（个性、有条件）。普遍性寓于特殊性之中。 **不平衡性**：主要矛盾（支配地位、决定作用）；矛盾主要方面（决定事物性质）。 | **矛盾分析法**（一分为二）。 **具体问题具体分析**（活的灵魂）。 **两点论与重点论统一**：解决问题抓**主要矛盾**；分析性质/形势看**主要方面**（看主流）。 |
| **量变质变规律** | 揭示发展的**形式和状态** | **量变**：数量增减，未超“度”。 **质变**：根本性质变化。 **关系**：量变是质变的必要准备；质变是量变的必然结果；相互渗透。 | 重视量的积累（积少成多）；不失时机促成飞跃；坚持**适度原则**（过犹不及）。 |
| **否定之否定规律** | 揭示发展的**方向和道路** | **辩证否定**：事物的自我否定，实质是“**扬弃**”（既克服又保留）。 **过程**：螺旋式上升，波浪式前进。 **特点**：前进性与曲折性的统一。 | 道路是曲折的，前途是光明的（如“梅花香自苦寒来”）。 |

#### **5\. 认识论**

*   
  **实践：** 人类能动地改造世界的社会性客观物质活动。具有直接现实性、自觉能动性、社会历史性 。  
* **实践与认识的辩证关系：**  
  *   
    **实践决定认识：** 实践是认识的**来源**、**动力**、**目的**、**唯一检验标准** 。  
  *   
    **认识反作用于实践：** 正确的认识促进实践，错误的认识阻碍实践 。  
* **认识的辩证过程：**  
  *   
    **第一次飞跃：** 感性认识（感觉、知觉、表象） $\\rightarrow$ 理性认识（概念、判断、推理） 。  
  *   
    **第二次飞跃：** 理性认识 $\\rightarrow$ 实践（更重要） 。  
* **真理：**  
  *   
    **定义：** 标志主观与客观相符合 。  
  *   
    **属性：** **客观性**（根本属性）、绝对性、相对性（具体性、条件性） 。

#### **6\. 唯物史观（历史唯物主义）**

*   
  **基本问题：** 社会存在与社会意识的关系。**社会存在决定社会意识** 。  
  *   
    **社会存在：** 物质生产方式（决定力量）、自然地理环境、人口因素 。  
  *   
    **社会意识：** 具有**相对独立性**（发展不同步、反作用） 。  
* **社会基本矛盾（根本动力）：**  
  * 生产力决定生产关系；生产关系反作用于生产力。  
  * 经济基础决定上层建筑；上层建筑（政治/观念）反作用于经济基础。**生产力是最终决定力量** 。  
*   
  **人民群众：** 历史的创造者，社会变革的决定力量（群众史观 vs 英雄史观） 。

### **二、马克思主义政治经济学**

#### **1\. 商品与货币**

* **商品二因素：**  
  *   
    **使用价值：** 自然属性，满足人的需要，价值的物质承担者 。  
  *   
    **价值：** 社会属性，本质属性，凝结在商品中无差别的人类劳动 。  
  *   
    *注：对立统一，不可兼得* 。  
* **劳动二重性：**  
  * **具体劳动：** 生产使用价值（自然属性）。  
  *   
    **抽象劳动：** 形成价值（社会属性，唯一源泉） 。  
*   
  **价值量：** 由**社会必要劳动时间**决定。与社会劳动生产率成**反比** 。  
*   
  **价值规律：** 商品交换以价值量为基础，实行等价交换。表现形式是**价格围绕价值上下波动** 。  
* **货币：** 本质是**一般等价物**。  
  * **基本职能：** 价值尺度（观念上的货币）、流通手段（一手交钱一手交货）。  
  *   
    **其他职能：** 贮藏手段（足值金属）、支付手段（赊销/工资/租金，钱货分离）、世界货币 。

#### **2\. 剩余价值与资本**

*   
  **剩余价值：** 雇佣工人创造的、被资本家无偿占有的超过劳动力价值的那部分价值 。  
* **资本分类：**  
  *   
    **不变资本（C）：** 生产资料，价值量不增殖 。  
  *   
    **可变资本（V）：** 劳动力，创造剩余价值 。  
* **生产方法：**  
  *   
    **绝对剩余价值：** 延长工作日 。  
  *   
    **相对剩余价值：** 缩短必要劳动时间（全社会劳动生产率提高） 。

### **2\. 常见命题陷阱与易错点（避坑指南）**

* **主体/主导置换：**

### **3\. 答题与思维技巧**

*   
  **首尾对比法：** 在排序题（如宪法修改排序）中，先确定最先发生的（开放度最小）或最后发生的（开放度最大），利用排除法快速解题 。

  * 看到“最本质特征”、“法治之魂”、“最高政治领导力量”，选**党的领导** 。

  * 看到“判断性质”、“看主流”，对应**矛盾主要方面** 。
