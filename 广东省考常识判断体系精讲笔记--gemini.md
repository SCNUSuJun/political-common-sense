# **广东省考常识判断体系精讲笔记**

## **第一部分：历史常识体系**

### **二、 古代政治与制度演变**

#### **1\. 中央与地方制度**

*   
  **西周**：实行**分封制** 。  
* **秦汉**：  
  * 中央设**三公九卿制**：丞相（百官之长，处理政事）、太尉（主管军事）、御史大夫（监察，下达诏令） 。  
  * 地方设**郡县制** 。  
* **隋唐**：  
  * 中央设**三省六部制**：中书省（草拟颁布诏令）、门下省（审核政令）、尚书省（执行政令，下设吏、户、礼、兵、刑、工六部） 。  
*   
  **宋朝**：二府三司制。设枢密使（军权）、参知政事（行政）、三司使（财政）以分割宰相权力 。  
* **元朝**：  
  * 中央：**中书省**为最高行政机构 。  
  * 地方：**行省制**（中国省制的开端） 。  
* **明朝**：  
  * 中央：明太祖**废除丞相**；明成祖确立**内阁**制度 。  
  * 地方：废行省，设三司（承宣布政使司、提刑按察使司、都指挥使司） 。  
*   
  **清朝**：雍正帝设立**军机处**，标志着**君主专制制度发展到顶峰** 。

#### **2\. 选官制度演变**

*   
  **先秦**：世卿世禄制（看血缘） 。  
*   
  **战国**：军功爵制（商鞅变法，看军功） 。  
*   
  **两汉**：察举制和征辟制（自下而上推荐，看品行/孝廉） 。  
*   
  **魏晋南北朝**：九品中正制（看门第，"上品无寒门，下品无士族"） 。  
*   
  **隋唐至清末**：**科举制** 。  
  * 隋炀帝：始建进士科，科举制正式形成 。  
  * 武则天：始创武举、殿试、糊名制 。  
  * 明清程序：  
    1.   
       **院试**：考中称“生员”（**秀才**） 。  
    2.   
       **乡试**（秋闱）：考中称“**举人**”，第一名**解元** 。  
    3.   
       **会试**（春闱）：考中称“**贡士**”，第一名**会元** 。  
    4.   
       **殿试**：皇帝主持，前三名分别为**状元、榜眼、探花** 。  
    *   
      *连中三元*指解元、会元、状元 。  
  * 废除：清末新政期间（1905年光绪年间） 。

#### **3\. 赋税与改革制度**

| 改革名称 | 时期/人物 | 核心内容与考点 |
| :---- | :---- | :---- |
| **初税亩** | 春秋（鲁国） | 按土地数量征税，承认土地私有 。 |
| **管仲改革** | 春秋（齐桓公） | “相地而衰征”（根据土地贫瘠程度征税），加速瓦解奴隶制 。 |
| **商鞅变法** | 战国（秦孝公） | 废井田开阡陌（土地私有）；奖励军功；户籍什伍制；连坐法；徙木立信 。 |
| **北魏孝文帝改革** | 南北朝 | 均田令；迁都洛阳；汉化政策（改汉姓、穿汉服、通汉婚） 。 |
| **租庸调制** | 隋-唐前期 | 以人头为主（租：田租；调：人头税；庸：纳绢代役） 。 |
| **两税法** | 唐朝后期 | 户税+地税，夏秋两季征收（人头税并入土地税，按资产和田亩征收） 。 |
| **王安石变法** | 北宋（宋神宗） | 青苗法（打击高利贷）、方田均税法、保甲法、农田水利法；目的富国强兵 。 |
| **一条鞭法** | 明朝（张居正） | 赋役合并，折银征收，人头税分摊入田亩，实物税转货币税 。 |
| **摊丁入亩** | 清朝（雍正） | 丁银全部摊入地亩，标志着延续数千年的**人头税彻底废除** 。 |

#### **2\. 边疆治理机构**

* **西藏**：  
  * 元朝：**宣政院**（管理佛教和西藏事务，西藏正式归属中央） 。  
  * 清朝：理藩院（管理蒙藏等少数民族）；雍正设**驻藏大臣** 。  
* **新疆**：  
  * 西汉：**西域都护府**（新疆正式归属中央的开始） 。  
  * 唐朝：安西都护府（唐太宗）、北庭都护府（武则天） 。  
  * 清朝：伊犁将军 。  
* **台湾**：  
  * 元朝：**澎湖巡检司**（管理琉球/台湾） 。  
  * 明末清初：郑成功收复台湾（驱逐荷兰殖民者）；清康熙年间统一台湾 。

### **四、 中国近代史**

#### **1\. 列强侵华与不平等条约**

| 战争 | 时间 | 条约 | 核心内容 | 影响 |
| :---- | :---- | :---- | :---- | :---- |
| **鸦片战争** | 1840-1842 | **《南京条约》** | 割**香港岛**；赔款2100万银元；开放广厦福宁上五口；协定关税 。 | 近代**第一个**不平等条约；**开始沦为**半殖民地半封建社会 。 |
| **第二次鸦片战争** | 1856-1860 | 《天津/北京条约》 | 增开天津为口岸；割九龙司地方一区给英国 。 |  **进一步加深**半殖民地化；俄国侵占领土最多 。 |
| **甲午中日战争** | 1894-1895 | **《马关条约》** | 割辽东、台湾全岛、澎湖；赔白银2亿；**允许在通商口岸设厂** 。 | 资本输出为主；**大大加深**半殖民地化 。 |
| **八国联军侵华** | 1900-1901 | **《辛丑条约》** | 赔白银4.5亿；东交民巷设使馆区（国人不得入）；严禁反帝 。 | 清政府沦为工具；**完全陷入**半殖民地半封建深渊 。 |

#### **2\. 救亡图存运动**

*   
  **开眼看世界**：林则徐（第一人，翻译《四洲志》）；魏源（《海国图志》，“师夷长技以制夷”） 。  
* **洋务运动**（19世纪60-90年代）：  
  * 口号：“自强”（军事）、“求富”（民用）；指导思想：中学为体，西学为用 。  
  * 代表：中央奕䜣；地方曾国藩（安庆内军械所-最早）、李鸿章（江南制造总局-最大）、张之洞（汉阳铁厂）、左宗棠（福州船政局） 。  
*   
  **戊戌变法**（1898）：康有为、梁启超“公车上书”揭序幕；设立**京师大学堂**（近代第一所国立大学） 。  
* **辛亥革命**（1911）：  
  * 组织：兴中会（第一个革命团体），同盟会（第一个资产阶级政党，纲领：驱除鞑虏，恢复中华，创立民国，平均地权） 。  
  * 核心：三民主义（民族、民权、民生），核心是**民权** 。  
  * 成果：推翻清朝，结束两千多年封建君主专制 。

#### **3\. 中共党史与革命历程**

* **建党初期**：  
  *   
    **中共一大**（1921.7）：上海→嘉兴南湖。标志建党。代表毛泽东、董必武；**陈独秀、李大钊未参加** 。  
  *   
    **中共二大**（1922）：制定最高和最低纲领，**首次提出反帝反封建** 。  
  *   
    **中共三大**（1923）：决定国共合作（党内合作） 。  
  *   
    **国民党一大**（1924）：新三民主义，标志**第一次国共合作正式形成** 。  
  *   
    **中共四大**（1925）：提出无产阶级领导权和工农联盟 。  
* **土地革命**：  
  *   
    **南昌起义**（1927.8.1）：武装反抗第一枪，建军开始 。  
  *   
    **八七会议**（1927.8.7）：纠正陈独秀右倾，毛泽东提出“**枪杆子里出政权**” 。  
  *   
    **三湾改编**：确立党对军队的绝对领导 。  
  *   
    **古田会议**（1929）：思想建党、政治建军 。  
  *   
    **遵义会议**（1935）：纠正王明左倾，是**生死攸关的转折点**，确立毛泽东领导地位 。  
* **抗日战争**：  
  *   
    **九一八事变**（1931）：局部抗战开始 。  
  *   
    **七七事变**（1937）：全面抗战开始 。  
  *   
    **西安事变**（1936）：和平解决标志着抗日民族统一战线**初步形成** 。  
  * 关键战役：淞沪会战（粉碎三月灭亡计划）；**平型关大捷**（抗战以来首次大捷）；台儿庄战役（正面战场最大胜利）；**百团大战**（中共主动出击最大规模） 。  
  * 胜利日：9月3日 。  
* **解放战争**：  
  *   
    **三大战役**：辽沈、淮海（小推车推出来的，歼敌最多）、平津 。  
  *   
    **七届二中全会**（1949西柏坡）：工作重心由乡村转移到城市，提出“两个务必” 。  
  *   
    **渡江战役**：占领南京，国民政府覆灭 。

## **第二部分：文学与文化常识体系**

### **一、 文学流派与经典作品**

#### **1\. 诗歌发展脉络**

### **二、 传统文化常识**

#### **1\. 史书体例**

*   
  **纪传体**：以人物为中心。代表作\*\*《史记》\*\*（司马迁，第一部纪传体通史，被鲁迅誉为“史家之绝唱，无韵之离骚”） 。  
*   
  **编年体**：按时间顺序。代表作\*\*《春秋》**（第一部）、**《资治通鉴》\*\*（司马光，第一部编年体通史） 。  
*   
  **国别体**：按国家分。代表作《国语》（第一部）、《战国策》 。  
*   
  **断代史**：只记一代。代表作《汉书》（班固，第一部纪传体断代史） 。

#### **2\. 艺术成就**

* **书法**：  
  * 书圣：王羲之（东晋），代表作\*\*《兰亭序》\*\*（天下第一行书） 。  
  * 楷书四大家：欧阳询、颜真卿（颜筋）、柳公权（柳骨）、赵孟頫（元代） 。  
  * 草圣：张旭（与怀素并称“颠张醉素”） 。  
  * 宋四家：苏、黄、米、蔡 。  
* **绘画**：  
  * 顾恺之（东晋）：《洛神赋图》、《女史箴图》 。  
  * 展子虔（隋）：《游春图》（现存最古老山水画） 。  
  * 阎立本（唐）：《步辇图》（松赞干布求亲） 。  
  * 吴道子（唐）：**画圣**，“吴带当风” 。  
  * 张择端（北宋）：**《清明上河图》**（描绘汴京繁华） 。  
  * 黄公望（元）：《富春山居图》 。

## **第三部分：科技常识体系**

### **一、 物理常识**

#### **1\. 力学**

*   
  **牛顿第一定律**（惯性定律）：物体保持静止或匀速直线运动，除非有外力。应用：刹车人前倾、拍打衣服灰尘、跳远助跑 。  
*   
  **牛顿第三定律**：作用力与反作用力（大小相等、方向相反）。应用：划船、火箭升空、起跑蹬地 。  
* **摩擦力**：  
  * 减小摩擦：加润滑油、变滑动为滚动（轴承）。  
  * 增大摩擦：鞋底/轮胎花纹、涂防滑粉 。  
*   
  **流体压强**：流速大压强小（伯努利原理）。应用：飞机机翼升力（上凸下平）、香蕉球、两船不能并排行驶 。  
* **杠杆原理**：  
  *   
    **省力杠杆**（动力臂\>阻力臂）：省力费距离。如：撬棍、扳手、钳子、开瓶器、指甲剪 。  
  *   
    **费力杠杆**（动力臂\<阻力臂）：费力省距离。如：镊子、筷子、钓鱼竿、理发剪、船桨 。  
  *   
    **等臂杠杆**：天平、定滑轮 。

#### **2\. 光学**

*   
  **直线传播**：影子的形成、小孔成像、日食月食、激光准直、坐井观天 。  
*   
  **反射**：镜子成像、水中倒影、潜望镜、杯弓蛇影 。  
*   
  **折射**：介质改变。如：筷子弯折、潭清疑水浅、海市蜃楼、凸透镜成像、彩虹 。  
*   
  **散射**：蓝天、朝霞、丁达尔现象 。  
* **不可见光**：  
  *   
    **红外线**：热效应强。应用：遥控器、夜视仪、热成像 。  
  *   
    **紫外线**：化学效应强。应用：杀菌、验钞、合成维生素D 。

#### **3\. 声学与热学**

* **声音**：  
  * 三要素：**音调**（频率决定高低）、**响度**（振幅决定大小）、**音色**（材料结构决定特质，闻其声知其人） 。  
  * 传播：真空**不能**传声。速度一般是固体\>液体\>气体（金属最快） 。  
  * 超声波（\>20kHz）：B超、声呐、清洗；次声波（\<20Hz）：预警、危害人体 。  
* **物态变化**：  
  * **熔化**（吸热）：固变液（冰化水）。  
  * **凝固**（放热）：液变固（结冰）。  
  * **汽化**（吸热）：液变气（蒸发、沸腾）。  
  *   
    **液化**（放热）：气变液（白气、露水、雾、“出汗”） 。  
  *   
    **升华**（吸热）：固变气（樟脑丸变小、干冰制造烟雾） 。  
  *   
    **凝华**（放热）：气变固（霜、雪、雾凇） 。

### **二、 化学与生物常识**

#### **1\. 化学物质**

* **气体**：  
  * 一氧化碳（CO）：有毒，煤气中毒，与血红蛋白结合 。  
  * 二氧化碳（CO2）：干冰（固态）用于人工降雨；温室效应 。

  * 含量排序：地壳（氧硅铝铁）；人体（氧碳氢氮钙）；海洋（氧氢氯钠） 。  
  * 金属：汞（液态、体温计）；钛（太空金属）；铁（应用最广）；钙（人体最多金属元素） 。

#### **2\. 生物与医学**

* **人体系统**：  
  * 消化：小肠是主要吸收场所；肝脏是最大消化腺/解毒器官 。

* **现代生物技术**：  
  * 基因工程：核心是DNA重组（转基因、胰岛素） 。

## **第四部分：生活常识与安全**

### **一、 急救与安全**

1.   
   **火灾**：湿毛巾捂口鼻、低姿前行、**严禁乘电梯**。精密仪器用二氧化碳灭火器；油类用泡沫；一般用干粉 。  
2.   
   **雷电**：室外找低洼处，**双脚并拢**下蹲（减少跨步电压），远离金属和树木 。  
3.   
   **中暑**：通风降温，补充淡盐水，**不可**快速大量饮水 。  
4.   
   **烫伤**：立即**冷水冲洗**，不可用冰块（防冻伤），不可挑破水泡 。  
5.   
   **骨折**：先固定再搬运 。  
6.   
   **扭伤**：24-48小时内**冷敷**（止血消肿），之后**热敷**（散瘀）。口诀：冷战热恋 。  
7.   
   **海姆里克法**：针对急性呼吸道异物堵塞（生命的拥抱） 。

### **二、 日常生活**

1. **食品安全**：  
   * 绿色食品：无污染、安全。  
   * 添加剂：防腐（苯甲酸钠）、膨松（碳酸氢钠/小苏打） 。  
   * 亚硝酸钠：致癌，防腐着色 。  
2. **家用电器**：  
   * 冰箱：制冷剂汽化吸热、液化放热。不宜太满（留空隙循环）。  
   * 微波炉：**不可用金属容器**（反射微波/打火）；加热带壳鸡蛋/盒装牛奶会爆炸 。  
3. **交通**：  
   * 高速爆胎：握紧方向盘，**点刹**（不可急刹）。  
   * 远光灯：照明差时用，会车时需切换近光。

### **3\. 做题实战经验**

*   
  **绝对化表述往往错误**：选项中出现“一定”、“凡是”、“所有”、“只有”等词汇，大概率是错误选项。例如“声音在固体中传播速度**一定**大于液体”是错的 。  
* **时间轴法排除**：  
  * 历史题注意朝代对应，如“宋江在玉米地斗剑”是错的（玉米明朝才传入） 。  
  * “关羽吃葡萄”是对的（葡萄西汉传入，关羽是汉末三国） 。  
* **常识反推法**：  
  * “下雪不冷化雪冷” —— 因为化雪是熔化/升华过程，需要**吸热**，导致环境温度降低 。  
  * “真金不怕火炼” —— 说明金的**化学性质稳定**（不易氧化），不是物理熔点高 。  
* **时政结合**：  
  * 关注\*\*“首次”、“第一”\*\*：如嫦娥六号（世界首次月背采样）、天问二号（小行星采样）。  
  * 关注**周年纪念**：如2025年是抗战胜利80周年，相关历史事件（九一八、七七事变、百团大战）即为高频考点 。
