# 广东省考政治理论与常识备考全面体系笔记





## 第一部分：广东省考政治理论考情综述





### 1. 考试结构与分值分布



- **题量构成**：2025年广东省考政治理论模块共考察 **10道题**。
- **题型分布**：
  - **判断题**：4道（难度较低，易拿分）。
  - **单选题**：4道（难度较低，主要考察基础理解）。
  - **多选题**：2道（难度极高，是拉分点，考察非常细致，甚至涉及考研政治深度的知识点）。
- **常识模块**：共考察 **5道题**，依然遵循广东省考一贯模式，重思维、重理解、重推理，整体难度不大，偏向好拿分。



### 2. 考察特点分析



- **紧贴大纲**：广东省考非常符合考试大纲要求，重点考察运用新思想（习近平新时代中国特色社会主义思想）的方法论来做题。
- **深度与广度**：
  - **政治理论**：考察内容“花”，种类多。判断和单选简单，多选极难（尤其是马哲部分）。
  - **常识判断**：偏向理解和推理，死记硬背的比重相对较小。
- **复习侧重**：不需要像考行政执法卷那样死扣法律条文细节，也不需要像其他省份那样大量背诵新修法规。核心在于“领导人讲话”和“基础政治逻辑”。

## 第二部分：政治理论核心知识体系





### 一、习近平新时代中国特色社会主义思想





#### 1. 核心方法论：“两个结合”与“六个坚持”



这是广东卷出题的核心逻辑，不仅考原文，还考运用。

- **两个结合**：
  - **第一结合**：马克思主义基本原理同中国具体实际相结合。
  - **第二结合**：马克思主义基本原理同**中华优秀传统文化**相结合（这也是习近平文化思想的重要组成部分，推进马克思主义文化理论创新的重大成果）。
- **六个坚持（世界观和方法论）**：
  - 必须坚持人民至上。
  - 必须坚持自信自立（对应：坚定信仰、信念，坚定四个自信，既不刻舟求剑也不照抄照搬）。
  - 必须坚持守正创新（对应：守正不迷失方向、创新把握时代；尊古不复古、接受新事物）。
  - 必须坚持问题导向。
  - 必须坚持系统观念。
  - 必须坚持胸怀天下。



#### 2. 新质生产力（高频考点）



- **定义**：创新起主导作用，摆脱传统经济增长方式、生产力发展路径，具有高科技、高效能、高质量特征，符合新发展理念的先进生产力质态。
- **核心要素**：**科技创新**（能够催生新产业、新模式、新动能）。
- **基本内涵**：劳动者、劳动资料、劳动对象及其优化组合的跃升。
- **核心标志**：全要素生产率大幅提升。
- **本质**：先进生产力。
- **特点**：创新。
- **关键**：质优。
- **绿色发展关系**：绿色发展是高质量发展的底色，新质生产力本身就是绿色生产力。



#### 3. 核心理念辨析

| -------------- | ------------------------------------------ | ------------------------------------------------------------ |
| **人民至上**   | 坚持以人民为中心，一切为了人民，依靠人民。 | 选拔干部、制定政策、解释历史（如脱贫攻坚）的根本出发点。     |
| **国家安全**   | 中国式现代化行稳致远的重要基础。           | 涉及国家生存发展的根本性表述，通常为正确选项。               |
| **耐心资本**   | 专注于**长期投资**的资本形式。             | 区别于中短期投资，强调“长线”思维（如投腾讯、阿里几十年不动）。 |

### 二、马克思主义哲学（马哲）





#### 1. 马克思主义的根本特征



- **科学性**：是马克思主义理论区别于其他理论的**根本特征**。
- **特征统一**：在实践基础上，科学性与革命性的统一。



#### 2. 唯物辩证法（难点）



- **核心内容**：唯物辩证法是马克思主义世界观和方法论的核心内容。
- **核心地位**：**矛盾分析法**在唯物辩证法的方法论体系中居于核心地位。
- **本质**：批判的和革命的。
- **思维能力方法**：分析与综合、归纳与演绎、抽象与具体、逻辑与历史等，都属于提高辩证思维能力的方法。
- **易错深坑（考研级难度）**：
  - **错误表述**：“唯物辩证法是客观辩证法在人的思维中的反映”。
  - **正确理解**：**主观辩证法**才是客观辩证法在人的思维中的反映。唯物辩证法包含客观辩证法和主观辩证法。



### 三、中共党史与奋斗历程





#### 1. 建党初期与国际关系



- **第一国际/第二国际/第三国际**：
  - 中国共产党成立时（一大），主要与**第三国际**（共产国际）建立紧密关系。
  - 一大闭幕时高呼“第三国际万岁”。
  - **排除法**：第二国际在一战期间（1914-1918）已式微，中国早期马克思主义运动主要受俄国十月革命（第三国际背景）影响，需与第二国际的社会民主主义划清界限。



#### 2. 历史时期与战略划分



| **历史时期**                           | **时间跨度** | **关键特征/战略**                              | **易错点辨析**                                               |
| -------------------------------------- | ------------ | ---------------------------------------------- | ------------------------------------------------------------ |
| **新民主主义革命时期**                 | 1919-1949    | 第一次国共合作、土地革命、抗日战争、解放战争。 | 第一次国共合作属于此时期（正确）。                           |
| **社会主义革命时期**                   | 1949-1956    | 三大改造。                                     | -                                                            |

| **改革开放和社会主义现代化建设新时期** | 1978-2012    | 经济建设为中心，改革开放。                     | 提出“三步走”战略。                                           |
| **中国特色社会主义新时代**             | 2012-至今    | 全面建设社会主义现代化强国。                   | “新发展阶段”属于此范畴。                                     |



#### 3. 党的基本路线



- **社会主义初级阶段基本路线**：核心内容是“一个中心、两个基本点”（以经济建设为中心，坚持四项基本原则，坚持改革开放）。



### 四、法律与道德关系



- **关系定位**：法律是成文的道德，道德是内心的法律。法治和德治不可分离、不可偏废。
- **上层建筑**：思想道德和法律都属于上层建筑的重要组成部分（观念上层建筑和政治上层建筑）。
- **相互作用**：
  - 思想道德为法治提供思想指引和价值基础。
  - 法治为思想道德提供制度支撑和法律保障。
  - **错误观点**：“法律不是调节人们思想的手段”（法律通过规范行为也会对思想产生影响和调节作用，不能绝对割裂）。



### 五、广东省情与高质量发展



- **发展历程**：从“三来一补”起步 -> 相对落后的农业省 -> 牵动全球的世界工厂 -> **国家重要先进制造业高地和发展新质生产力的主阵地**（2025高质量发展大会提法）。
- **重点会议**：关注每年年初的“广东省高质量发展大会”和“广东省政府工作报告”。

## 第三部分：常识与政治理论解题/蒙题逻辑体系





### 一、核心思维：公务员思维（政治站位）



这是做所有常识和政治题的底层逻辑。

1. **以人民为中心（服务型政府）**：
   - **判断标准**：凡是体现为人民服务、方便群众、保障人民利益、倾听人民呼声的选项，通常是正确的。凡是增加群众负担、推诿责任、侵犯群众利益的，通常是错误的。
   - **案例应用**：
     - *行政处罚*：执法人员不得当场收缴罚款（为了程序正义），但100元以下为了便民可以当场收缴。若选项说“100元以上不得当场收缴”，则是为了规范权力，保护人民，是正确的。
     - *乡村振兴*：要求农民“无条件配合”撤并村庄 -> 错误（侵犯权益，非服务型）。
     - *污染赔偿*：化工厂证明未超标排放就不赔偿？ -> 错误（只要造成损害，尤其是举证责任倒置保护弱势群体，不能简单免责）。
2. **政治正确性（涉及党和领袖）**：
   - **命题人视角**：命题人政治站位极高，不会在涉及党、国家领导人（习、马、列、毛、邓等）、核心指导思想（马克思主义）的选项上轻易挖坑或设置负面描述。
   - **技巧**：
     - 出现“习近平总书记指出”、“党中央强调”的选项，大概率是正确的。
     - 选项内容若是歌颂党、国家伟大成就、民族自豪感的，通常选对。
     - **领袖与事迹对应**：题目中出现领袖名字（如毛泽东指挥），通常不会张冠李戴去考查领袖名字错了，而是默认为真。



### 二、具体蒙题技巧与陷阱识别





#### 1. 民族自豪感与国家优秀论



- **逻辑**：体现我国在国际上领先、独立自主、打破封锁、历史性成就的选项，往往是正确的。
- **关键词**：世界第一、最大、首次、自主研发（如北斗、空间站、高铁）。
- **反向逻辑**：如果选项说我国要“超越西方”、“把超越美国作为主要目标”，通常是错误的。**中国式现代化是走自己的路，不屑于跟别人比，不搞霸权争夺**。



#### 2. 绝对化与中庸之道



- **绝对词**：选项中出现“必须”、“一定”、“只能”、“绝对”、“任何”等词汇，往往是错误的（除非是政治原则，如“必须坚持党的领导”）。
- **中庸词**：选项中出现“可以”、“一般”、“可能”、“不一定”、“补充设定”等留有余地的词汇，往往是正确的。



#### 3. 逻辑陷阱识别



- **张冠李戴（找双胞胎/多胞胎）**：
  - **定义**：命题人喜欢把相近或相反的概念互换。
  - **常见对子**：
    - 供给侧 <-> 需求侧
    - 上层 <-> 下层（如：食肉鱼类在下层，选项说上层即错）
    - 东 <-> 西（如：东江纵队/西江纵队）
    - 生 <-> 死
    - 主观 <-> 客观
    - 第一国际/第二国际 <-> 第三国际
  - **破题**：看到一个概念，马上联想它的对立面或相似面，检查是否被置换。
- **数字陷阱**：
  - 选项中出现具体数字（如年代、数量、第几次、罚款金额）极易挖坑。
  - **策略**：如果无法确定，优先怀疑带数字的选项。
- **偷换概念**：
  - 将政府职能偷换为社会机构（如：全过程人民民主案例中，将“人大代表”、“居委会”偷换为“直播平台网友投票”，后者非国家政治生活，不属于全过程人民民主范畴）。
- **时间逻辑（历史题）**：
  - **越早越不可能**：历史题中，技术、制度、文化现象通常随时间发展而复杂化。如果选项说很早的朝代（如秦汉）出现了很成熟的晚期事物（如京剧、完善的市舶司制度），通常是错的。
- **语意重复/同源**：
  - 正确选项往往与题干在用词、语意、逻辑上最顺畅、最契合。
  - 题干中的字在某个选项中重复出现，该选项可能为真（如：题干讲“纪纲”，选项有“纪纲一废”）。



#### 4. 法律题的特殊蒙法



- **原则**：虽然是法律题，但依然遵循“为人民服务”和“常理”。
- **案例**：
  - 社保缴纳：工伤和生育保险由单位交，个人不交（常识：出事了单位负责）。失业、医疗、养老需要个人承担一部分。
  - 新法新规：通常涉及保护弱势群体、规范政府权力、促进便民。

## 第四部分：讲师李铁的政治理论实战经验与精华总结



本部分提炼自讲师在授课过程中的口语化经验之谈，是避坑和高效复习的关键。



### 1. 备考策略与取舍（非常重要）



- **政治理论复习范围**：
  - **核心圈**：过去一年**总书记的重要讲话** + **新思想（习近平新时代中国特色社会主义思想）概论**。
  - **马哲**：广东考得很难、很偏（如主观/客观辩证法），甚至达到考研政治深度。**建议**：如果基础不好，不要为了那一两个难点题去啃肖秀荣的考研书，**性价比极低，可以战略性放弃深难点**，只抓基础（如实践、矛盾）。
  - **法律法规**：广东卷**不需要**像其他省份那样大量复习新修法律法规文件，也不考查偏门的红头文件。重点是《求是》杂志里总书记的文章和年度重要会议（如三中全会）。
  - **省情**：考前突击一下当年的**广东省政府工作报告**和**高质量发展大会**即可（通常考1题，且较简单）。



### 2. 考试难度与心态



- **难度分层**：单选和判断题非常简单，基本拉不开分；多选题很难，是拉分关键。
- **心态建设**：政治理论不用花太多时间死磕细节，题目相对常规。遇到不会的题（特别是常识），不要慌，用“蒙题技巧”去推导。



### 3. 快速记忆与解题直觉



- **古语引用题**：这类题实质是**言语理解题**。不要被古文吓倒，提取关键词与选项对应即可。
  - *例子*：题干讲“志之所趋”（志向），选项找与“志向/信念”有关的词（如“信念坚定”），无需完全读懂古文。
- **领导人引用的话**：
  - 如果考查某句话是谁引用的或填空：
    - **杜甫**的诗句常被引用来表达对民生的关切（“安得广厦千万间”）。
    - **孙中山**的话（如“家给人足”）常被引用，体现历史传承和统战思维。
- **科技常识**：
  - 广东爱考**科技成就**。
  - **深海探测器区分**：
    - **奋斗者号、深海勇士号、蛟龙号** = **载人**潜水器。
    - **梦想号** = 深海科考船（钻探），**不是**载人潜水器。
    - **海基一号** = 导管架。
    - **海牛II号** = 钻探器。



### 4. 核心避坑指南



- **不要死记硬背**：常识复习浩如烟海，死记硬背“神舟X号”所有细节效率极低。要遵循“二八原则”，只看最重要、最体现国家实力的。
- **相信直觉**：在考场上，如果有“民族自豪感”涌上心头，或者觉得某个选项“为人民服务”得特别丝滑，选它，不要犹豫。
- **多选题技巧**：多选题很难全对，但可以利用**排除法**。只要发现一个选项有明显的逻辑漏洞（如违背人民利益、张冠李戴），就果断排除，提高准确率。
- **时政时效性**：如果当年的省考在政府工作报告发布之前考，则看前一年的重要会议精神；如果在之后考，则看当年的。



### 5. 实战口诀（提炼）



- “心中有党，考试不慌。”
- “有上就有下，有东就有西，看到一边想另一边（双胞胎思维）。”
- “涉及党和领袖，默认正确；涉及外国对比，中国最牛。”
- “为了钱损害人民利益的（如减少福利、乱征地），一律选错。”
