# **广东省考常识判断与政治理论备考体系笔记**

## **第一部分：广东省考考情与备考策略**

### **一、 模块分布与特点**

根据2025年广东省考真题情况分析，常识部分分为两个主要模块：

1. **政治理论（10题）：**  
   * **题型结构：** 4个判断题、4个单选题、2个多选题。  
   * **难度分析：** \* 判断题和单选题相对简单，拉分空间小。  
     * 多选题难度较大（尤其是马哲部分）。  
   * **内容侧重：** 领导人过去一年的重要讲话、马克思主义哲学（马哲）、新时代中国特色社会主义思想（新思想）。  
2. **常识判断（5题）：**  
   * **特点：** 重思维、重理解、重推理，不是单纯死记硬背。  
   * **内容侧重：** 高新技术、省情（如广东省高质量发展大会）、法律常识（社保法等）、二十届三中全会精神。

### **二、 核心备考方向**

| 备考板块 | 重点内容与资料来源 | 备注 |
| :---- | :---- | :---- |
| **领导人讲话** | 过去一年的重要讲话、求是杂志（第一篇）。 | 重点关注高频词汇与新提法。 |
| **新思想概论** | “两个结合”、“六个坚持”、中国式现代化。 | 通用性强，行政执法卷也适用。 |
| **马克思主义哲学** | 唯物辩证法、矛盾分析法、主观/客观辩证法。 | 广东考查较深，可参考考研政治资料。 |
| **高新技术** | 深海探测、航空航天、新能源。 | 广东重视科技成就考查。 |
| **省情省况** | 政府工作报告、全省高质量发展大会。 | 关注年初会议及下半年省委书记讲话。 |

## **第二部分：政治理论核心知识点梳理（基于真题）**

### **一、 习近平文化思想与新思想方法论**

**1\. “两个结合”**

* **第一结合：** 马克思主义基本原理同中国具体实际相结合。  
* **第二结合：** 马克思主义基本原理同**中华优秀传统文化**相结合（真题判断题考点）。

**2\. “六个坚持”与立场观点方法**

* **内容：** 坚持人民至上、坚持自信自立、坚持守正创新、坚持问题导向、坚持系统观念、坚持胸怀天下。  
* **对应关系辨析（真题案例）：**  
  * **自信自立：** 对应“坚定信仰、坚定信念、四个自信（道路、理论、制度、文化）”。  
  * **守正创新：** 对应“不迷失方向（守正）”、“把握时代、把握未来（创新）”、“尊古不复古、接受新事物”。

### **二、 新质生产力**

| 维度 | 核心内容 |
| :---- | :---- |
| **三新（催生）** | 新产业、新模式、新动能。 |
| **核心要素** | **科技创新**（真题判断题考点）。 |
| **特征** | 高科技、高效能、高质量。 |
| **核心标志** | 全要素生产率大幅提升。 |
| **本质** | 先进生产力。 |
| **底色** | 绿色发展（新质生产力本身就是绿色生产力）。 |

### **三、 马克思主义哲学（高难点）**

**1\. 科学性特征**

* 马克思主义区别于其他理论的**根本特征**是**实践性**，而非单纯的科学性（真题判断题考点，需注意区分）。  
* 马克思主义是科学性与革命性的统一，建立在实践基础之上。

**2\. 唯物辩证法（多选难点）**

* **核心地位：** 矛盾分析法在唯物辩证法方法论体系中居于核心地位。  
* **本质：** 批判的和革命的。  
* **主观与客观辩证法（深层考点）：**  
  * **客观辩证法：** 外部世界本身的辩证运动。  
  * **主观辩证法：** 客观辩证法在人的思维中的反映（**注意：** 不是唯物辩证法是客观辩证法在思维中的反映，范围不能搞错）。  
* **思维方法：** 分析与综合、归纳与演绎、抽象与具体等均属于提高辩证思维能力的方法。

### **四、 党史与奋斗历程（多选考点）**

**1\. 国际关系**

* **第三国际：** 中国共产党成立时（一大）强调与**第三国际**建立紧密联系，而非第二国际。  
* **第二国际：** 在一战期间已式微，主要涉及社会民主主义，与中共早期直接关联较少。

**2\. 历史时期划分**

* **新民主主义革命时期（1949年前）：** 包含第一次国共合作（国民党一大）、三大（决定个人名义加入国民党）。  
* **社会主义革命与建设时期（1949-1978）：** 包含三大改造等。  
* **改革开放和社会主义现代化建设新时期（1978-2012）：** 提出“三步走”战略（邓小平理论）、“一个中心、两个基本点”（初级阶段基本路线）。

## **第三部分：常识判断重点与解题思维**

### **一、 经济与二十届三中全会精神**

**1\. 耐心资本**

* **定义：** 专注于**长期投资**的资本形式（真题考点）。  
* **特征：** 不追求短期回报，陪伴企业成长（如孙正义投阿里、南非报业投腾讯）。

**2\. 粮食安全（基本农作物）**

* **国家层面基本农作物：** 粮食（米袋子）、油料（油罐子）、蔬菜（菜篮子）。  
* **排除项：** 苗木（不属于满足基本生产生活需要的国家层面基本农作物）。

### **二、 法律常识（社会保险法）**

1\. 费用缴纳主体区分

| :--- | :--- | :--- |  
| 基本养老保险 | 单位 \+ 个人 | 个人需承担部分。 |  
| 基本医疗保险 | 单位 \+ 个人 | 个人需承担部分。 |  
| 失业保险 | 单位 \+ 个人 | 个人需承担部分。 |

### **三、 广东省情（高质量发展）**

**1\. 发展历程描述**

* 起步：“三来一补”。  
* 发展：相对落后的农业省 \-\> 牵动全球的世界工厂。  
* **当前定位（2025省情考点）：** 引领全球的**智造基地**和**新兴产业重要策源地**（注意区分：不是“新质生产力主阵地”这种泛词，而是具体的“智造”与“策源地”）。

### **四、 科技常识辨析**

**1\. 深海探测装备**

* **“梦想号”：** 深海科考船（钻探），不是载人潜水器。  
* **载人潜水器代表：** “奋斗者”号、“深海勇士”号、“蛟龙”号。

## **第四部分：常识做题方法论（蒙题技巧与命题人视角）**

此部分为解题的核心逻辑工具，用于在知识盲区进行有效推断。

### **一、 公务员思维（适当积累与逻辑判断）**

**核心逻辑：** 站在政府和公务员的角度思考，核心思想是\*\*“以人民为中心”**和**“建设服务型政府”\*\*。

**应用场景分析：**

1. **军队服务：** 军队全面停止有偿服务（因为军队本质是为人民服务，不能以赚钱为目的）。  
2. **征地与权益：**  
   * 征地补偿方案需经**村民会议**讨论（听取人民意见）。  
   * 村委会决定侵害村民权益，村民可申请撤销（保护人民合法权益）。  
3. **举证责任：** 环境污染案中，若要求受害村民举证因果关系，是增加了人民负担，违背服务理念。因此，举证责任通常倒置或由企业承担（**注意：** 法律规定中环境侵权因果关系往往由排污者举证，或者至少不能让弱势村民承担过重举证责任）。  
4. **行政处罚与便民：**  
   * **当场收缴罚款：** 小额罚款（如100元以下）可以当场收缴，这是为了**便民**（减少群众折腾），而非为了“收钱”。如果是大额（如100元以上），为了规范防止腐败，通常不能当场收缴。  
   * **组织听证：** 降低资质等级属于重罚，**应当**组织听证（保障当事人权益）。

### **二、 命题人视角（政治站位）**

**核心逻辑：** 命题人政治站位极高，涉及党、国家领导人、核心指导思想的选项，通常是正确的，不会在这些严肃内容上“挖坑”。

1. **领导人与理论：**  
   * 涉及马列毛邓三科习的选项，往往是对的。  
   * 出现领导人名字（如毛泽东指挥某战役、习近平引用某典故），若非极明显的张冠李戴，通常默认正确。  
   * **技巧：** 如果选项中有领导人讲话内容与所涉省区对应（如习总书记在山西讲黄花致富），通常全对，因为命题人不会制造领导人“讲错话”或“张冠李戴”的错误印象。  
2. **中庸之道（语气判断）：**  
   * **绝对化词汇：** “必须”、“绝对”、“一定”——往往是错误的（除了涉及党的领导、主权等原则问题）。  
   * **中庸词汇：** “不一定”、“可以”、“逐步”、“可能”——往往是正确的。  
   * **示例：** “法律法规未规定的，地方性法规**可补充**设定”——语气中庸，符合立法补充机制，通常正确。

### **三、 民族自豪感（国家优秀论）**

**核心逻辑：** 体现国家成就、技术突破、民族优秀的选项，往往是正确的。

1. **适用领域：**  
   * **航空航天：** 空间站（“我们自己的”）、北斗导航、火星探测（祝融）、太阳探测（羲和）。  
   * **基础设施：** 高铁（世界最大网）。  
   * **重大赛事：** 冬奥会（氢燃料电池车应用）。  
2. **避坑指南（反向逻辑）：**  
   * **不要盲目“超越”：** 中国式现代化不追求“超越西方发达国家”（不与人争，走自己的路），也不遵循“世界各国共同模式”（我们是**中国式**）。  
   * **投入与产出：** “研发经费支出跃居世界第一”可能不仅是错的，而且不符合“低投入高产出”或“人才红利”的自豪感逻辑（实际上我国经费总量可能未第一，但人才多）。

### **四、 命题人挖坑套路识别**

| 挖坑手段 | 识别特征与应对策略 |
| :---- | :---- |
| **夸张与绝对** | 出现“最”、“第一”、“全面停止”、“绝对”等词。需警惕，除非是众所周知的公认第一（如高铁网）。 |
| **数字陷阱** | 选项中出现具体数字（如3名航天员、第5次反围剿、1930年、100元）。**数字是高频挖坑点**，若不确定，优先怀疑数字选项。 |
| **偷换概念** | 将“政府职责”偷换给“企业”或“社会机构”。 例如：全过程人民民主的主体应是人大、政府、居委会，而非“直播平台”（商业机构）。 |
| **张冠李戴（双胞胎）** | **核心技巧：有上就有下，有东就有西。** 1\. **相对概念：** 如果选项提到“上层”，思考是否有“下层”（如肉食性青鱼在下层而非上层）。 2\. **供需关系：** 题目问“需求侧”，选项出现“企业生产”（供给侧），则该选项错误。 3\. **地理方位：** 题目说“文昌发射”，考虑是否其实是“酒泉”或“西昌”。 4\. **成对出现：** 两个选项长得很像（如张桂梅和王兰花的事迹互换），通常错误就在这两个选项中（互为干扰项）。 |
| **时空错乱** | **历史题逻辑：越早越不可能发生。** 1\. **技术限制：** 秦朝不可能有“舶来品”（西汉张骞通西域后才有）、明朝不可能唱“京剧”（清朝才形成）。 2\. **朝代对应：** 记住关键时间节点（如长征背景是第五次反围剿失败，而非第一次）。 |

### **五、 逻辑连贯与语感**

**技巧：** 问题和正确选项之间往往本就是一句话拆开的。

* **找重复字：** 正确选项中包含的关键词，往往在题干中出现过，或者与题干的逻辑气场最吻合。  
* **顺眼原则：** 读起来最顺畅、逻辑最严密、符合常理（如“党纪国法不能成橡皮泥”对应“纪纲一废，何事不生”）的选项。

## **第五部分：作者经验之谈（实战精华）**

本部分提取自作者（李铁、弓似月弦长）的授课口语及实战经验，属于“避坑”与“抄近道”的智慧。

### **一、 高频考点与复习建议**

1. **政治理论复习边界：**  
   * **不用学太深：** 广东的政治理论考得比较浅（除个别马哲题），不需要像考研那样死磕细节。  
   * **资料范围：** 重点看26年国考政治理论备考资料，加上广东省过去一年领导人讲话即可。信用法规等其他省份考的内容，广东一般不考。  
   * **战略性放弃：** 遇到极难的马哲题（如主观辩证法与客观辩证法的深层关系），如果复习成本太高，可以战略性放弃，因为这类题大家都不会，拉不开分。  
2. **常识复习性价比：**  
   * **不要死记硬背：** 尤其是科技常识，不要背几大页的神舟飞船数据，考的概率极低。不如掌握“蒙题思维”。  
   * **省情突击：** 考前半小时突击一下省政府工作报告和高质量发展大会的核心词汇（如“智造基地”），性价比最高。

### **二、 易混知识点区分技巧**

1. **社会保险：** 记住“工伤”和“生育”是公司全包，个人不掏钱。失业保险虽然听起来惨，但个人还得交。  
2. **历史事件排序：**  
   * **三步走战略：** 属于改革开放时期，不是社会主义建设时期（1956-1978）。  
   * **党史：** 党的一大对应第三国际；新民主主义革命时期包含第一次国共合作。  
3. **概念对立（双胞胎法）：**  
   * 看到“酸性”想“碱性”（治理盐碱地）。  
   * 看到“供给侧”想“需求侧”（扩大消费是需求，企业改革是供给）。  
   * 看到“上层”想“下层”（鱼类活动水层）。

### **三、 考场实战心态与操作**

1. **心态建设：**  
   * “心中有党，考试不慌”。遇到不懂的题，先看哪个选项政治站位高、符合主流价值观。  
   * 常识题不要纠结，每道题控制在30秒-40秒。不会做是正常的，大家都蒙，你只要蒙的逻辑比别人科学就能赢。  
2. **排除法应用：**  
   * **找不同：** 如果三个选项都是正面的、歌颂的，一个选项是中性的或略带限制的，或者三个是深海的，一个是浅海的，选那个“不合群”的。  
   * **看字数与重合度：** 某些题目中，正确选项往往是重复题干字眼最多、表述最完整、最符合逻辑通顺度的那一个。  
3. **特定题型秒杀：**  
   * **领导人说话题：** 看到习近平总书记说的话、引用的典故，选项中解释其含义的，找那个最能体现“为人民服务”、“家国情怀”或“历史唯物主义”的选项。  
   * **法律题：** 哪怕没学过法，只要选项符合“保护弱势群体（人民）”、“程序正义（听证）”、“便民原则”，大概率是对的。

**总结：** 广东省考常识与政治理论备考核心在于\*\*“抓大放小”\*\*。政治理论抓核心文件与领导人讲话，常识判断抓底层逻辑（公务员思维、民族自豪感）与解题技巧。不要陷入死记硬背的汪洋大海，要用逻辑推理去弥补知识盲区。
