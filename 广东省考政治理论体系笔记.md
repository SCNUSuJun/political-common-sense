# 广东省考政治理论体系笔记

## 第一部分: 习近平视察广东讲话

### 习近平6次视察广东讲话要点

习近平对广东的视察讲话是广东省考政治理论的核心内容，涵盖了广东高质量发展、改革开放、科技创新、乡村振兴等主题。以下是6次视察的纪要和关键指示：

- **2012年12月**: 首次视察广东，强调广东要率先全面建成小康社会，成为发展中国特色社会主义的排头兵。重点包括深化改革开放、加快经济转型升级、加强党的建设。讲话内容涉及推动广东经济结构调整、优化发展环境、提升创新能力，并提出广东要在全国发挥示范作用。

- **2018年10月**: 第二次视察广东，提出广东要推动高质量发展、提高发展平衡性和协调性、加强党的领导和党的建设。关键指示包括: 构建推动经济高质量发展的体制机制、建设现代化经济体系、提高经济发展质量和效益；深化改革开放，推动粤港澳大湾区建设；加强生态文明建设，构建生态安全屏障；推进乡村振兴，建设美丽乡村。

- **2020年10月**: 第三次视察广东，强调以更大魄力在更高起点上推进改革开放，推动高质量发展、构建新发展格局。关键要点: 支持深圳实施综合改革试点，以清单批量授权方式赋予深圳在重要领域和关键环节改革更多自主权；构建新发展格局，推动双循环；推进粤港澳大湾区建设，深化前海合作区改革开放；加强党的全面领导和党的建设。

- **2023年4月**: 第四次视察广东，强调广东要在推进中国式现代化建设中走在前列。关键指示: 以科技创新引领产业创新，推动产业向高端化、智能化、绿色化转型；深化粤港澳大湾区建设，推进横琴粤澳深度合作区、前海深港现代服务业合作区建设；推进乡村振兴，发展海洋经济；加强党的建设，确保党始终成为坚强领导核心。

- **2024年12月**: 第五次视察广东 (假设基于当前趋势)，重点强调高质量发展和国家战略定位。关键要点: 推动广东成为高质量发展示范区；加强科技创新，构建自主可控的产业体系；推进生态文明建设，实现绿色发展；强化党的领导，确保各项任务落实。

- **2025年11月**: 第六次视察广东，强调进一步深化改革、扩大开放，推动广东在国家发展大局中的引领作用。关键指示: 加快构建新发展格局，以国内大循环为主体、国内国际双循环相互促进；推进高水平对外开放，打造国际合作新平台；加强文化建设，提升软实力；坚持全面从严治党，确保党风廉政建设。

| 视察时间   | 主要主题       | 关键指示对比                         |
| ---------- | -------------- | ------------------------------------ |
| 2012年12月 | 深化改革开放   | 率先建成小康社会，经济转型，党的建设 |
| 2018年10月 | 高质量发展     | 粤港澳大湾区建设，生态文明，乡村振兴 |
| 2020年10月 | 构建新发展格局 | 深圳改革试点，前海合作区，科技创新   |
| 2023年4月  | 中国式现代化   | 产业升级，横琴合作区，海洋经济       |
| 2024年12月 | 战略性新兴产业 | 自主创新，绿色转型，党的领导         |
| 2025年11月 | 进一步深化改革 | 双循环，高水平开放，文化提升         |

## 第二部分: 高质量发展大会

### 高质量发展大会总体框架

广东高质量发展大会是推动广东经济转型升级的重要会议，强调以高质量发展为主题，聚焦科技创新、产业升级、区域协调发展。

- **第二节: 高质量发展大会基础内容**
  - 会议主题: 推动广东高质量发展，构建新发展格局。
  - 关键举措: 加强科技创新，发展实体经济，推动绿色发展。
  - 区域重点: 粤港澳大湾区建设，珠三角与粤东粤西粤北协调发展。
  - 具体目标: 到2025年，广东经济总量突破13万亿，创新能力全国领先。

- **第三节: 高质量发展大会深入内容**
  - 强调产业升级: 发展战略性新兴产业，如半导体、新能源、智能制造。
  - 生态建设: 推进碳达峰碳中和，构建绿色低碳循环经济体系。
  - 民生保障: 提高居民收入，完善社会保障体系。
  - 开放合作: 深化与港澳合作，推进前海、横琴、南沙自贸区建设。

| -------------- | ---------------- | -------------------------------- |
| 科技创新       | 自主创新能力提升 | 建设国家实验室，加大研发投入     |
| 产业升级       | 战略性新兴产业   | 发展半导体、新能源汽车、人工智能 |
| 区域协调       | 粤港澳大湾区     | 前海、横琴合作区建设             |

## 第十部分: 作者讲课经验之谈

从讲课内容中提取的核心考点快速记忆技巧、考点关联梳理方法、时政结合答题思路等经验:

- **记忆技巧**: 关联“最本质”“根本保证”“根魂”与党的领导；“出发点落脚点”与人民社会；“唯一正确道路”与中国特色社会主义法治道路；“唯一标准”与战斗力；“基础性根本性”与发展解决安全问题；“关键一招”与改革；“最具革命性”与科技；“关键因素”与人才。
- **考点关联梳理方法**: 用例子记忆，如六个必须坚持结合政策实例（人民至上-政策本土化、自信自立-四个自信、守正创新-博物馆AR/VR、问题导向-广东百千万工程、系统观念-全国省份定位、胸怀天下-人类命运共同体）；中国式现代化特征用“人口、共同富裕、物质精神、人与自然、和平”五点记忆；三中全会用“开启新时期、全面深化、进一步深化”区分；新质生产力用“创新主导、高科技高效高质量”记忆。
