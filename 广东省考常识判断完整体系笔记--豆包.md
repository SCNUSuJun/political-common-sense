# 广东省考常识判断完整体系笔记--豆包

## 一、物理常识

### （一）力学基础

#### 1. 牛顿定律

| 定律名称                 | 核心内容                                               | 典型应用 / 现象                                        |
| ------------------------ | ------------------------------------------------------ | ------------------------------------------------------ |
| 牛顿第一定律（惯性定律） | 一切物体总保持匀速直线运动或静止状态，除非外力迫使改变 | 纸飞机离手后继续飞行、刹车时乘客前倾、拍打法除衣服灰尘 |
| 牛顿第二定律             | 加速度与作用力成正比，与质量成反比，方向与作用力一致   | -                                                      |
| 牛顿第三定律             | 作用力与反作用力大小相等、方向相反、作用在同一直线     | 划船拨水、起跑蹬地、火箭喷气起飞                       |
| 万有引力定律             | 两质点间引力与质量乘积成正比，与距离平方成反比         | -                                                      |

#### 2. 常见的力

| 力的类型    | 核心定义                                             | 关键特点 / 应用                                              |
| ----------- | ---------------------------------------------------- | ------------------------------------------------------------ |
| 压力 / 压强 | 压力是垂直作用在物体表面的力；压强 = 压力 / 受力面积 | 增大压强：增大压力 / 减小受力面积；流体流速越大压强越小；大气压随高度升高降低 |
| 摩擦力      | 阻碍物体相对运动 / 趋势的力，方向与相对运动方向相反  | 减小摩擦：滑动改滚动、减小接触面粗糙程度 / 压力；增大摩擦：鞋底花纹、防滑粉 |
| 重力        | 地球吸引产生的力                                     | 纬度越高重力加速度越大；超重（向上加速）、失重（向下加速 / 太空） |
| 浮力        | 浸在液体中的物体受向上浮力，大小等于排开液体重力     | 小昆虫水面行走、沉船打捞                                     |

#### 3. 杠杆原理

| 杠杆类型 | 核心特点                   | 典型示例                             |
| -------- | -------------------------- | ------------------------------------ |
| 省力杠杆 | 动力臂＞阻力臂，省力费距离 | 铡刀、手推车、撬棍、扳手、开瓶器     |
| 费力杠杆 | 动力臂＜阻力臂，费力省距离 | 钓鱼竿、镊子、筷子、起重机吊臂、扫帚 |
| 等臂杠杆 | 动力臂 = 阻力臂            | 天平、跷跷板、定滑轮                 |

#### 4. 物态变化

| 变化类型 | 状态转换  | 吸放热 | 典型现象                             |
| -------- | --------- | ------ | ------------------------------------ |
| 熔化     | 固态→液态 | 吸热   | 冰融成水、雪消                       |
| 凝固     | 液态→固态 | 放热   | 水结冰、“瀚海阑干百丈冰”             |
| 汽化     | 液态→气态 | 吸热   | 水蒸发、湿衣服晾干                   |

#### 5. 热力学应用

| 知识点     | 核心内容                                                     | 典型应用                                                     |
| ---------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 传热方式   | 热传导：直接接触传热；热对流：流体宏观运动传热；热辐射：电磁辐射传热（真空中最有效） | 晒太阳（热辐射）、开水变凉（热传导）、加盖开水凉得慢（热对流） |
| 分子运动论 | 分子不停息无规则热运动，温度越高越剧烈                       | 气味扩散、二手烟、面包房飘香                                 |
| 热胀冷缩   | 物体受热膨胀、遇冷收缩                                       | 路面留缝、电线下垂、水银体温计、鸡蛋冷水中易剥壳             |

### （二）光学

#### 1. 光的传播

| 传播形式 | 核心定义                         | 典型应用 / 现象                                        |
| -------- | -------------------------------- | ------------------------------------------------------ |
| 直线传播 | 同种均匀介质中沿直线传播         | 影子、小孔成像、日食 / 月食、立竿见影、一叶障目        |
| 反射     | 分界面改变方向返回原物质         | 镜子、水面倒影、杯弓蛇影、镜中花水中月                 |
| 折射     | 斜射入另一透明介质，传播方向改变 | 水中筷子弯折、潭清疑水浅、海市蜃楼、彩虹、鱼缸看鱼变大 |
| 散射     | 通过不均匀介质，部分光偏离原方向 | 晴天天空蓝、雾天光线朦胧、丁达尔现象                   |

#### 2. 可见光与不可见光

| 类别              | 核心特征                                                     | 应用 / 特点                                                  |
| ----------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 可见光            | 人眼可感知，波长递减 / 频率递增：红橙黄绿青蓝紫              | 红光：波长最长、衍射 / 穿透强，用于警示；黄光：穿透强、人眼敏感，用于路灯 / 雾灯 |
| 不可见光 - 红外线 | 波长长于可见光                                               | 红外探测器（感应门、遥控器）、热成像仪、加热干燥             |
| 不可见光 - 紫外线 | 波长短于可见光                                               | 杀菌、验钞、促进维生素 D 合成；过量照射有害                  |
| 电磁波谱          | 波长由短到长：γ 射线、X 射线、紫外线、可见光、红外线、无线电波 | 不可见光真实存在，可传递能量、传播、反射                     |

#### 3. 光的三原色与颜料三原色

| 类型         | 原色组成   |
| ------------ | ---------- |
| 光的三原色   | 红、绿、蓝 |
| 颜料的三原色 | 红、黄、蓝 |

### （三）声学

| 衡量维度 | 决定因素           | 典型应用             |

- 速度：常温下固体＞液体＞气体，金属中更快

#### 3. 声音的分类

| 类型   | 频率范围   | 特点                                                       | 应用                         |
| ------ | ---------- | ---------------------------------------------------------- | ---------------------------- |
| 可听声 | 20Hz~20kHz | 人耳可感知                                                 | -                            |

| 次声波 | ＜20Hz     | 传播远、穿透力强、方向性差；易与人体器官共振，有伤害性     | 灾害预警、核爆监测           |

## 二、化学常识

### （一）非金属物质

| 元素 / 化合物 | 核心特征                                               | 应用 / 注意事项                                              |
| ------------- | ------------------------------------------------------ | ------------------------------------------------------------ |
| 氢（H）       | 宇宙含量最多、元素周期表首个元素；氢气是最轻气体，易燃 | -                                                            |
| 碳（C）       | 形成化合物最多、构成细胞最基本元素                     | 一氧化碳（CO）：易致煤气中毒；二氧化碳（CO₂）：干冰用于冷藏、人工降雨、灭火 |
| 氮（N）       | 空气中最多、农作物肥料三要素之一                       | 氮气（N₂）：食品防腐、安全气囊；液氮：制冷剂                 |
| 氧（O）       | 人体 / 海洋 / 地壳中含量最多；除金外金属均能与其反应   | 臭氧（O₃）：吸收紫外线，存在于臭氧层                         |
| 硫（S）       | 二氧化硫（SO₂）：造成酸雨，有还原性、漂白性            | 酸雨：pH＜5.6，主因 SO₂和氮氧化物                            |
| 氯（Cl）      | 氯气（Cl₂）：浅黄绿色、有毒，氧化能力强                | 自来水消毒                                                   |
| 硅（Si）      | 半导体材料                                             | 硅：制半导体、太阳能电池；二氧化硅（SiO₂）：制光导纤维、玻璃 |

### （二）金属物质

| 金属 / 合金 | 核心特征                                        | 应用 / 注意事项                                              |
| ----------- | ----------------------------------------------- | ------------------------------------------------------------ |
| 钛（Ti）    | 耐高温 / 低温、抗酸碱、高强度低密度（太空金属） | 人体植入物、飞机 / 火箭结构件                                |
| 铁（Fe）    | 应用最广泛；能与水、氧气反应（双吸剂）          | 钢：含碳 0.02%~2.04%；不锈钢：含铬 12%~30%                   |
| 铜（Cu）    | 紫红色、导热导电好，人类最早使用的金属          | 青铜（最早合金）、黄铜（铜锌）、白铜（铜镍）                 |
| 钙（Ca）    | 生物必需，人体含量最多的无机盐                  | 氧化钙（生石灰）：干燥剂、建筑材料；氢氧化钙（熟石灰）：中和酸性土壤 |
| 汞（Hg）    | 俗称水银，常温液态，熔点最低                    | 玻璃体温计（热胀冷缩）                                       |
| 铅（Pb）    | 耐蚀、质软、熔点低、有毒                        | 爆米花 / 松花蛋易含铅，过量致中毒                            |

### （三）酸、碱、盐

#### 1. 酸碱性与 pH 值

- pH＜7：酸性，值越小酸性越强；pH＞7：碱性，值越大碱性越强；pH=7：中性
- 酸雨：pH＜5.6

#### 2. 酸、碱、盐核心要点

| 类别 | 代表物质             | 核心特征 / 应用                                     |
| ---- | -------------------- | --------------------------------------------------- |
| 酸   | 盐酸（HCl）          | 胃酸主要成分，用于除锈、清洁                        |
|      | 硝酸（HNO₃）         | 氧化性强，能溶解多数金属及合金                      |
|      | 浓硫酸（H₂SO₄）      | 油状液体，氧化性 / 脱水性强；稀释需注酸入水并搅拌   |
| 碱   | 氢氧化钠（NaOH）     | 火碱 / 烧碱，用于干燥、炼油、造纸                   |
|      | 氢氧化钙（Ca (OH)₂） | 消石灰 / 熟石灰，检验 CO₂、中和酸性土壤、制波尔多液 |
| 盐   | 碳酸钠（Na₂CO₃）     | 苏打 / 纯碱，清洁油污、制玻璃 / 陶瓷                |
|      | 碳酸氢钠（NaHCO₃）   | 小苏打，食品膨松剂                                  |
|      | 氯化钠（NaCl）       | 食盐主要成分                                        |
|      | 亚硝酸钠（NaNO₂）    | 腌渍食品中含量高，有毒，易生成致癌亚硝胺            |
|      | 碳酸镁（MgCO₃）      | 干燥剂、增加手掌与器械摩擦力（运动员用）            |

#### 3. 酸碱鉴别

| 类别 | 石蕊溶液 | 酚酞溶液 | 反应特征                                                    | 电离特征         |
| ---- | -------- | -------- | ----------------------------------------------------------- | ---------------- |
| 酸   | 变红     | 无色     | 与碱 / 碱性氧化物反应生成盐和水；与某些金属反应生成盐和氢气 | 阳离子全部是 H⁺  |
| 碱   | 变蓝     | 变红     | 与酸 / 酸性氧化物反应生成盐和水                             | 阴离子全部是 OH⁻ |

### （二）生物代谢与行为

#### 1. 新陈代谢

- 同化作用（合成代谢）：合成有机物、贮存能量（如光合作用）
- 异化作用（分解代谢）：分解有机物、释放能量（如呼吸作用）
- 关系：异化作用释放的能量供同化作用使用

#### 2. 光合作用

- 主体：植物、藻类、某些细菌
- 条件：可见光、光合色素
- 过程：将 CO₂（或 H₂S）和水转化为有机物，释放 O₂（或 H₂）
- 影响因素：光照、CO₂浓度、温度、水分

#### 3. 呼吸作用

| 类型     | 条件   | 产物                                                         | 典型现象                         |
| -------- | ------ | ------------------------------------------------------------ | -------------------------------- |
| 有氧呼吸 | 有氧气 | 水、CO₂                                                      | -                                |
| 无氧呼吸 | 缺氧   | 乳酸（动物、乳酸菌、马铃薯块茎等）；酒精 + CO₂（酵母菌、多数植物） | 人剧烈运动后肌肉酸痛；糯米酿米酒 |

#### 4. 蒸腾作用

- 主体：植物（主要是叶子）
- 过程：水分以水蒸气状态散失到大气
- 影响因素：光照、温度、湿度、风

### （四）灾害应对

#### 1. 火灾

- 逃生：有序疏散，勿乘电梯；低姿匍匐，湿毛巾捂口鼻；身上着火就地打滚
- 灭火器：对准火苗根部；不同灭火器适用场景见下表

| 灭火器类型     | 灭火剂           | 适用火灾                                 | 不适用火灾                             |
| -------------- | ---------------- | ---------------------------------------- | -------------------------------------- |
| 二氧化碳灭火器 | 液态 CO₂         | 图书档案、精密仪器、电气设备、油类初起火 | 过氧化物火灾                           |
| 泡沫灭火器     | 硫酸铝、碳酸氢钠 | 固体 / 汽油 / 柴油火灾                   | 水溶性可燃液体、电气设备、精密仪器火灾 |
| 干粉灭火器     | 无机盐           | 可燃固体 / 液体 / 气体、带电设备初起火   | 金属燃烧火灾                           |

#### 2. 雷电

- 室外：低洼处藏身，低头曲身，远离金属 / 电力设施，不持金属物品
- 室内：不接打电话，拔电器插头，不开门窗

#### 3. 洪水

- 转移至山坡 / 高地；来不及则爬屋顶 / 大树等待救援

#### 4. 泥石流

- 向与泥石流垂直方向的两侧山坡跑，远离沟道 / 河谷

#### 5. 地震

- 低矮楼层：疏散至开阔空地
- 高层：躲在低矮牢固家具下 / 小开间（厨房、卫生间），护头部

## 八、授课经验与解题技巧

### （一）高频考点快速识别技巧

1. 物理考点：光的传播（直线 / 反射 / 折射）、物态变化（吸放热）、声音的传播（真空不能传声）、牛顿定律（惯性 / 作用力反作用力）为高频，题干中出现 “影子”“倒影”“弯折”“白气”“真空” 等关键词可快速定位考点。
2. 化学考点：常见物质（CO₂、NaOH、甲烷）、酸碱盐（pH 值、酸雨）、有机化合物（酒精、甲醛）为高频，题干中出现 “酸雨”“消毒”“膨松剂”“有毒气体” 等可定位。
3. 生物考点：人体系统（内分泌 / 免疫）、营养物质（维生素缺乏症）、传染病分类（甲类鼠疫霍乱）、遗传物质（DNA/RNA）为高频，题干中出现 “糖尿病”“夜盲症”“疫苗”“基因” 等可定位。
4. 历史考点：古代政治制度（三省六部、行省制）、选官制度（科举制）、近代不平等条约（《南京条约》《辛丑条约》）、党史（遵义会议、抗战战役）为高频，题干中出现 “丞相”“科举”“条约”“会议” 等可定位。
5. 文化考点：古代思想（儒家 / 道家）、传统节日（诗词对应）、四大发明、书法绘画（代表作对应）为高频，题干中出现 “仁政”“元宵”“活字印刷”“《兰亭序》” 等可定位。

### （二）易混知识点区分方法

1. 物理易混点：
   - 光的反射（镜子、倒影）vs 折射（弯折、海市蜃楼）：看是否 “穿过介质”，穿过则折射，返回则反射。
   - 超声波（＞20kHz，B 超）vs 次声波（＜20Hz，灾害预警）：记频率范围 + 典型应用。
   - 物态变化吸放热：“熔升汽吸，凝液固放”（熔化、升华、汽化吸热；凝华、液化、凝固放热）。
2. 化学易混点：
   - 苏打（碳酸钠）vs 小苏打（碳酸氢钠）：记用途（苏打清洁、小苏打膨松）。
   - 一氧化碳（有毒，煤气中毒）vs 二氧化碳（无毒，干冰）：记毒性 + 应用。
   - 有机化合物：甲烷（天然气）、乙烯（催熟）、乙醇（酒精）、甲醛（污染）：记核心用途。
3. 历史易混点：
   - 科举制：隋炀帝创进士科，武则天创武举 / 殿试，明清八股取士，1905 年废除。
   - 近代条约：《南京条约》（首个）、《马关条约》（资本输出）、《辛丑条约》（完全沦为）：记核心影响。
   - 党史会议：八七会议（枪杆子里出政权）、遵义会议（转折点）、七届二中全会（乡村转城市）：记核心口号 / 意义。
4. 文化易混点：
   - 楷书四家：欧颜柳赵（欧阳询、颜真卿、柳公权、赵孟頫），赵孟頫为元朝，其余为唐。
   - 四大名旦：梅尚程荀（梅兰芳、尚小云、程砚秋、荀慧生），无杨小楼。
   - 史书体裁：编年体（按时间）、纪传体（按人物）、国别体（按国家）：记核心分类依据。

### （三）时政考点结合记忆思路

1. 高新技术考点（探月、载人航天、行星探测）：按 “时间线 + 核心成就” 记忆，如嫦娥系列（一号到八号）、神舟系列（五号首次载人、七号首次出舱），结合最新进展（2025 年天问二号、神舟二十号）。
2. 新能源考点：结合我国能源政策，记 “核能（华龙一号、石岛湾）、太阳能、风能” 的应用，区分可再生 / 不可再生能源。
3. 党史考点：结合周年纪念（如抗战胜利 80 周年），记关键事件时间（九一八 1931、七七 1937、抗战胜利 1945）。

### （四）常识题快速排除错误选项的经验

1. 绝对化表述排除：选项中出现 “一定”“所有”“唯一”“只有” 等绝对化词汇，大概率错误（如 “声音一定能在固体中传播”“所有金属都能和氧反应”，注意例外：金不与氧反应、声音传播需介质）。
2. 常识性错误排除：结合生活常识排除，如 “微波炉可用金属容器”“烫伤用冰块冷敷”“汽车加油可打手机” 等明显违背生活常识的选项。
3. 时间 / 朝代错误排除：如 “唐朝活字印刷”“宋朝玉米种植”“元朝行省制不是省制开端” 等，记准关键事件的朝代 / 时间。
4. 概念混淆排除：如 “光年是时间单位”“纯碱是碱”“超声波是听不到的声音，次声波是听得到的” 等，区分易混概念的本质。
5. 选项矛盾排除：若两个选项表述矛盾，必有一个错误，结合考点进一步判断（如 “红外线可杀菌” vs “紫外线可杀菌”，记紫外线杀菌、红外线加热）。
