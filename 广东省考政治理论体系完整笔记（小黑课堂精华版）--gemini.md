## 第一篇：马克思主义基本原理



本篇涵盖马克思主义哲学、政治经济学和科学社会主义，是政治理论的基石。



### 第一章 马克思主义概述



**1. 诞生与构成**

- **诞生标志：** 1848年《共产党宣言》的发表。
- **翻译：** 1920年陈望道翻译首个中文全译本。
- **三大组成部分：** 马克思主义哲学、马克思主义政治经济学、科学社会主义。

**2. 理论来源与两大发现**

| **组成部分**       | **直接理论来源**   | **代表人物/思想**                      | **备注**                                   |
| ------------------ | ------------------ | -------------------------------------- | ------------------------------------------ |
| **马克思主义哲学** | 德国古典哲学       | 黑格尔（辩证法）、费尔巴哈（唯物主义） | 吸收了辩证法的合理内核和唯物主义的基本内核 |
| **政治经济学**     | 英国古典政治经济学 | 亚当·斯密、大卫·李嘉图                 | 劳动价值论                                 |
| **科学社会主义**   | 英法空想社会主义   | 圣西门、傅立叶、欧文                   | 提供了思想素材                             |

- **马克思的两大发现：** 唯物史观、剩余价值学说（使社会主义由空想变为科学）。

**3. 鲜明特征**

- **理论特征：** 以实践为基础的科学性和革命性的统一。
- **理论品质：** 与时俱进。
- **活的灵魂：** 具体问题具体分析。
- **根本方法：** 理论联系实际。

### 第二章 马克思主义哲学



1. 哲学基本问题

哲学的基本问题是思维和存在的关系问题（即意识和物质的关系问题）。

| **问题维度** | **具体内容**                 | **派别划分** | **观点主张**                               |
| ------------ | ---------------------------- | ------------ | ------------------------------------------ |
| **第一方面** | **何者为第一性**（谁决定谁） | **唯物主义** | 物质决定意识（世界的本原是物质）           |
|              |                              | **唯心主义** | 意识决定物质（世界的本原是精神）           |
| **第二方面** | **有无同一性**（能否被认识） | **可知论**   | 世界是可以被认识的（马克思主义坚持可知论） |
|              |                              | **不可知论** | 世界是不能被认识或不能被完全认识的         |

**2. 唯物主义与唯心主义的历史形态**

- **唯物主义的三种形态：**
  - **古代朴素唯物主义：** 把物质归结为一种或几种具体的**实物**（如金、木、水、火、土、气）。
  - **近代形而上学唯物主义：** 把物质归结为**原子**；具有孤立、静止、片面的缺陷。
  - **现代辩证唯物主义和历史唯物主义：** 认为物质的唯一特性是**客观实在性**。
- **唯心主义的两种形态：**
  - **主观唯心主义：** 主张**人（主体）的精神**（如人的感觉、心理、意志）是世界的本原（例：心外无物、存在即被感知）。
  - **客观唯心主义：** 主张**脱离人之外的客观精神**（如上帝、理、道、绝对观念）是世界的本原（例：理在事先、道生一）。

**3. 唯物论（物质与意识）**

- **物质：** 标志客观实在的哲学范畴。**客观实在性**是物质的唯一特性。
- **运动：** 物质的根本属性和存在方式。**静止**是运动的特殊状态（相对的）。世界是绝对运动和相对静止的统一。
- **意识：** 人脑对客观存在的主观映象。意识具有**主观能动性**（自觉能动性）。
  - **辩证关系：** 物质决定意识；意识对物质具有能动的反作用（正确的意识促进发展，错误的意识阻碍发展）。
  - **发挥主观能动性的前提：** 尊重客观规律。

**4. 唯物辩证法（两大特征：联系与发展）**

- **普遍联系：**
  - **特征：** 客观性（自在事物和人为事物的联系都是客观的）、普遍性、多样性、条件性。
  - **整体与部分：** 整体居主导地位，统率部分；部分影响整体，**关键部分**甚至对整体功能起决定作用（系统观念体现整体性）。
- **永恒发展：**
  - **实质：** 新事物的产生和旧事物的灭亡。
  - **判断标准：** 是否符合事物发展的必然趋势（而非出现时间的先后或力量的强弱）。

**5. 唯物辩证法的三大规律**

| **规律名称**       | **地位/作用**                              | **核心内容/方法论**                                          |
| ------------------ | ------------------------------------------ | ------------------------------------------------------------ |
| **对立统一规律**   | **实质和核心**，揭示了发展的**源泉和动力** | **矛盾的基本属性：** 同一性（相互依存、贯通）、斗争性（相互排斥、分离）。 **普遍性与特殊性：** 普遍性即共性（无条件），特殊性即个性（有条件）；普遍性寓于特殊性之中。 **两点论与重点论：** 解决问题抓**主要矛盾**（重点），分析性质看**矛盾主要方面**（主流）。 |
| **量变质变规律**   | 揭示了发展的**形式和状态**                 | **量变：** 数量增减，度的范围内；**质变：** 根本性质变化。 **关系：** 量变是质变的必要准备，质变是量变的必然结果。 **方法论：** 坚持适度原则，重视量的积累。 |
| **否定之否定规律** | 揭示了发展的**方向和道路**                 | **辩证否定：** 事物的自我否定，实质是**“扬弃”**（既克服又保留）。 **过程：** 螺旋式上升，波浪式前进。 **特点：** 前进性与曲折性的统一。 |

**6. 认识论**

- **实践：** 人类能动地改造世界的社会性的客观物质活动。
  - **特征：** 直接现实性（本质特征）、自觉能动性、社会历史性。
  - **基本形式：** 物质生产实践（最基本）、社会政治实践、科学文化实践。
- **实践与认识的辩证关系：**
  - 实践决定认识：来源、动力、目的、唯一检验标准。
  - 认识反作用于实践：正确认识促进，错误认识阻碍。
- **认识的过程：**
  - **第一次飞跃：** 感性认识（感觉、知觉、表象） $\rightarrow$ 理性认识（概念、判断、推理）。
  - **第二次飞跃：** 理性认识 $\rightarrow$ 实践（更重要）。
- **真理：** 标志主观与客观相符合。具有**客观性**（内容客观、标准客观）、**绝对性**和**相对性**（具体性、条件性）。

**7. 唯物史观（历史唯物主义）**

- **社会存在与社会意识：**
  - **根本分歧：** 唯物史观认为**社会存在决定社会意识**；唯心史观认为社会意识决定社会存在。
  - **社会存在：** 地理环境、人口因素、**物质生产方式**（决定力量）。
  - **社会意识：** 具有**相对独立性**（发展不同步、有反作用）。
- **社会基本矛盾（社会发展的根本动力）：**
  - **生产力 vs 生产关系：** 生产力决定生产关系，生产关系反作用于生产力。
  - **经济基础 vs 上层建筑：** 经济基础决定上层建筑，上层建筑反作用于经济基础。
    - *注：生产力是最终决定力量；政治上层建筑（国家政权）在观念上层建筑中居主导。*
- **人民群众：** 历史的创造者，社会变革的决定力量（群众史观）。

### 第三章 马克思主义政治经济学



**1. 商品**

- **含义：** 用来交换、能满足人的某种需要的劳动产品。
- **二因素：**
  - **使用价值：** 自然属性，价值的物质承担者。
  - **价值：** 社会属性，本质属性（凝结在商品中无差别的人类劳动）。
  - *关系：对立统一，不可兼得。*

**2. 劳动二重性**

- **具体劳动：** 生产使用价值，体现人与自然关系（自然属性）。
- **抽象劳动：** 形成价值，体现人与人关系（社会属性，唯一源泉）。
- *注意：是同一劳动的两个方面，不是两次劳动。*

**3. 价值量与价值规律**

- **价值量决定：** 由**社会必要劳动时间**决定（非个别劳动时间）。
- **关系：** 价值量与社会必要劳动时间成正比，与社会劳动生产率成反比。
- **价值规律：** 价格围绕价值上下波动（表现形式）。

**4. 货币**

- **本质：** 一般等价物。
- **职能：**
  - **价值尺度：** 衡量价值大小（观念上的货币，如标签）。
  - **流通手段：** 交换媒介（一手交钱一手交货，现实的货币）。
  - **支付手段：** 赊购赊销、还债、交租税、发工资（钱货分离）。
  - **贮藏手段：** 足值的金属货币。
  - **世界货币。**

**5. 剩余价值**

- **定义：** 雇佣工人创造的、被资本家无偿占有的超过劳动力价值的那部分价值。
- **资本分类（依据在剩余价值生产中的作用）：**
  - **不变资本（C）：** 生产资料，价值量不增殖。
  - **可变资本（V）：** 劳动力，创造剩余价值（包括劳动力价值+剩余价值）。
- **生产方法：**
  - **绝对剩余价值：** 延长工作日。
  - **相对剩余价值：** 缩短必要劳动时间（依靠全社会劳动生产率提高）。

  1. **人口规模巨大**的现代化。
  2. **全体人民共同富裕**的现代化。

### 第三章 总体布局（五位一体）





#### 第一节 经济建设：推动高质量发展



**1. 新发展阶段、理念与格局**

- **新发展阶段：** 全面建成小康社会之后，开启全面建设社会主义现代化国家新征程。
- **新发展理念：**
  - **创新：** 引领发展的第一动力（解决**发展动力**问题）。
  - **协调：** 内在要求（解决**发展不平衡**问题）。
  - **绿色：** 必要条件（解决**人与自然和谐**问题）。
  - **开放：** 必由之路（解决**发展内外联动**问题）。
  - **共享：** 本质要求（解决**社会公平正义**问题）。
- **新发展格局：** 以**国内大循环**为主体、国内国际双循环相互促进。
  - 关键：经济循环畅通无阻。
  - 最本质特征：实现高水平自立自强。

**2. 基本经济制度**

- **所有制：** 公有制为主体、多种所有制经济共同发展。
  - **公有制（主体）：** 国有经济（主导作用）、集体经济。
  - **非公有制：** 民营经济是生力军。**“两个毫不动摇”**。
- **分配制度：** 按劳分配为主体、多种分配方式并存。
  - 提高劳动报酬在**初次分配**中的比重。
  - 第三次分配：慈善公益（自愿性）。
- **体制：** 社会主义市场经济体制。
  - **市场**在资源配置中起**决定性作用**，更好发挥政府作用（有效市场+有为政府）。

**3. 现代化经济体系与战略**

- **乡村振兴：**
  - 总纲：农业强国。总抓手：乡村振兴战略。
  - 首要任务：保障粮食和重要农产品稳定安全供给。
  - 总要求：产业兴旺、生态宜居、乡风文明、治理有效、生活富裕。
- **区域协调发展：** 推进以人为核心的新型城镇化（首要任务：农业转移人口市民化）；推进以**县城**为重要载体的城镇化建设。



#### 第二节 政治建设：全过程人民民主



**1. 制度体系**

- **根本政治制度：** **人民代表大会制度**（实质：人民当家作主）。
- **基本政治制度：**
  - **政党制度：** 中国共产党领导的多党合作和政治协商制度。
    - 首要前提：坚持中共领导。
    - 方针：长期共存、互相监督、肝胆相照、荣辱与共。
    - 机构：人民政协（专门协商机构，主题是团结和民主）。
  - **民族区域自治制度：** 主线是铸牢**中华民族共同体意识**。
  - **基层群众自治制度：** 村委会、居委会（自我管理、自我教育等）。

**2. 全过程人民民主**

- **定位：** 社会主义民主政治的**本质属性**。
- **特征：** 全链条、全方位、全覆盖；最广泛、最真实、最管用。



#### 第三节 文化建设：建设文化强国



**1. 指导思想与目标**

- **根本制度：** 坚持马克思主义在意识形态领域指导地位。
- **目标：** 2035年建成文化强国。

连续性、创新性、统一性、包容性、和平性。



#### 第四节 社会建设：保障和改善民生

### 第四章 四个全面与重要保障



**1. 全面深化改革**

- **总目标：** 完善和发展中国特色社会主义制度、推进国家治理体系和治理能力现代化。
- **方向：** 坚持社会主义市场经济改革方向。
- **出发点落脚点：** 促进社会公平正义、增进人民福祉。

**2. 全面依法治国**

- **总目标：** 建设中国特色社会主义法治体系、建设社会主义法治国家。
- **核心要义/法治之魂：** 党的领导。
- **首要任务：** 依宪治国、依宪执政。
- **重点任务/主体工程：** 法治**政府**建设。
- **最后一道防线：** 公正司法。

**3. 全面从严治党**

- **地位：** 新时代党的建设的鲜明主题。
- **主线：** 加强党的长期执政能力建设、先进性和纯洁性建设。
- **统领/首位：** **政治建设**（首要任务：保证全党服从中央，坚持党中央权威和集中统一领导）。
- **根基：** 坚定理想信念宗旨。
- **治本之策：** 纪律建设。
- **反腐败：** 最彻底的自我革命（第二个答案）。

**4. 总体国家安全观**

- **关键：** “总体”。
- **五大要素：** 以人民安全为**宗旨**，以政治安全为**根本**（生命线），以经济安全为**基础**，以军事、科技、文化、社会安全为**保障**，以促进国际安全为**依托**。

**5. 国防和军队建设**

- **强军目标：** 听党指挥（灵魂）、能打胜仗（核心）、作风优良（保证），建设世界一流军队。
- **根本原则：** 党对人民军队的绝对领导（军委主席负责制是最高层次、统领地位）。
- **根本标准：** 战斗力。

**6. “一国两制”与祖国统一**

- **最高原则：** 维护国家主权、安全、发展利益。
- **前提基础：** “一国”。
- **台湾问题：**
  - 政治基础：一个中国原则和“九二共识”。
  - 实践途径：两岸和平发展、融合发展。
  - 根本动力：团结台湾同胞、争取台湾民心。

## 附录：小黑老师实战经验与“避坑”指南



本部分严格依据笔记中“考场传真”、“母题特训”及讲师批注整理，汇集了高频考点、易错陷阱与记忆技巧。



### 一、 核心概念“张冠李戴”陷阱



- **根本/基本类：**
  - **根本**政治制度是人民代表大会制度；**基本**政治制度包括政党制度、民族区域自治、基层群众自治。
  - **根本**保证是党的领导；**根本**任务是解放和发展生产力；**根本**目的是增进民生福祉。
- **主体/主导类：**
  - **主体**是人民/公有制/国内大循环；**主导**是国有经济/政治上层建筑。
  - 科技创新主体是**企业**（非高校，高校是生力军/主力军）。
- **决定/影响类：**
  - **生产力决定**生产关系（非反之）；**社会存在决定**社会意识。
  - 关键部分对整体起**决定**作用（特定条件下），而非只是影响。



### 二、 词语替换与程度陷阱



- **时间点混淆：**
  - 新时代是**全面建成小康社会之后**，而非之前。
  - 碳达峰是**2030**年前，碳中和是**2060**年前。
  - 基本实现现代化是**2035**年，建成强国是**本世纪中叶**。
- **程度词错误：**
  - 共同富裕 $\neq$ **同步**富裕、**同等**富裕。
  - 人工智能 $\neq$ 具有**意识**（只是机械物理过程，无自觉能动性）。
  - 规律可以**认识、利用**，但不能**创造、消灭、改变**。
- **顺序颠倒：**
  - **先**有法律规定，**后**有依法监督。
  - **市场**起决定性作用，更好发挥**政府**作用（不能反过来）。
  - **初次分配**重效率，**再分配**重公平（虽然现在强调兼顾，但功能侧重不同）。



### 三、 快速记忆与解题口诀



- **中国式现代化特色：** “人富精然走道路”（人口、富裕、精神、自然、和平）。
- **六个必须坚持：** “无问自创喜人”（胸怀天下、问题导向、自信自立、守正创新、系统观念、人民至上）。
- **价值规律作用：** “资生收”（调节资源、刺激生产力、调节收入）。
- **哲学判断技巧：**
  - **解决问题** $\rightarrow$ 抓主要矛盾。
  - **判断性质/形势** $\rightarrow$ 看矛盾主要方面（看主流）。
  - **古诗词题：** 找关键词对应原理（如“梅花香自苦寒来”对应事物发展的前进性与曲折性；“纸上得来终觉浅”对应实践是认识的来源）。



### 四、 命题人思维分析



- **结合时政：** 题目常以习近平总书记最新讲话或重要会议（如二十大、二十届三中全会）为背景，考察背后的理论依据（如“新质生产力”对应生产力与生产关系原理）。
- **首尾对比法：** 在排序题中，先确定首项或尾项，利用排除法快速解题。
- **绝对化表述：** 选项中出现“唯一”、“任何”、“完全”等绝对化词语时，往往是错误的（除特定政治原则外，如“唯一宗旨”）。
- **逻辑对应：**
  - 看到“最本质特征”选“党的领导”。
  - 看到“法治之魂”选“党的领导”。
  - 看到“生命线”在法治中指“政治安全”，在教育中指“高质量发展”。
