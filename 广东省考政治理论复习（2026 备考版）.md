# **广东省考政治理论复习（2026 备考版）**

## **第一部分：习近平总书记视察广东重要讲话（核心必考）**

**考情提示**：这是广东省考政治理论的“灵魂”，考察频次最高。重点掌握**历次视察的时间节点、视察地点、核心金句**以及**对广东的定位要求**。

### **1\. 广东的总定位与总要求**

* **三个角色**：广东是改革开放的**排头兵**、**先行地**、**实验区**。  
* **大湾区定位**：在大湾区建设中，广东是**主力军**、**火车头**。  
*   
  **经济大省责任**：在编制“十五五”规划时，要有高站位、大格局，体现**走在前、作示范、挑大梁**的责任担当 。  
*   
  **改革地位**：广东是在我国改革开放中**得风气之先**的地方 。

### **2\. 历次视察详细复盘（时间线逻辑）**

#### **(1) 2012年12月：改革开放的“起笔”**

* **背景**：十八大后习近平总书记首次离京视察，首站即广东。  
*   
  **核心金句**：**“改革不停顿，开放不止步”** 。  
* **关键评价**：  
  * 改革开放创造了举世瞩目的“广东奇迹”。  
  *   
    **深圳**是改革开放**最精彩的起笔** 。  
*   
  **要求**：敢于啃硬骨头，敢于涉险滩；尊重人民首创精神 。

#### **(2) 2018年10月：改革开放再出发**

* **主要行程**：珠海（横琴）、清远（英德）、深圳、广州。  
*   
  **核心主题**：**高举新时代改革开放旗帜** 。  
* **关键指示**：  
  *   
    **城市建设**：更多采用\*\*“微改造”\*\*这种“绣花”功夫，注重文明传承、文化延续，**让城市留下记忆，让人们记住乡愁**（切忌大拆大建） 。  
  *   
    **民营经济**：为民营企业、中小企业发展创造更好条件 。  
  *   
    **区域协调**：提高发展平衡性和协调性，把**汕头、湛江**作为重要发展极 。

#### **(3) 2020年10月：新征程新辉煌**

* **主要行程**：潮州、汕头。  
*   
  **核心主题**：以更大魄力、在更高起点上推进改革开放 。  
* **具体指示**：  
  *   
    **文化保护（潮州）**：保护好**广济桥**、潮州非遗。强调不能搞过度修缮、过度开发 。  
  *   
    **侨批文化（汕头）**：保护好\*\*“侨批”**文物（“侨批”记载了老一辈侨胞艰难的创业史和浓厚的家国情怀），推动全社会加强**诚信建设\*\* 。  
  *   
    **科技创新**：企业要聚焦国内短板产品，在自主研发上加倍努力 。

#### **(4) 2023年4月：中国式现代化的广东实践**

* **主要行程**：湛江、茂名、广州。  
*   
  **核心主题**：在推进中国式现代化建设中**走在前列** 。  
* **细分考点（极高频）**：  
  *   
    **海洋经济（大食物观）**：既向陆地要食物，也向海洋要食物。建设**海上牧场**、**“蓝色粮仓”** 。  
  *   
    **红树林保护（湛江）**：湛江红树林是\*\*“国宝”\*\*，要像爱护眼睛一样守护好 。  
  *   
    **交通枢纽**：**徐闻港**是连接粤港澳大湾区和海南自贸港的**现代化水陆交通运输综合枢纽**（黄金水道、咽喉要道） 。  
  *   
    **实体经济**：始终坚持以**制造业立省**，更加重视发展实体经济（不能脱实向虚） 。  
  *   
    **科技自立自强**：是中国式现代化建设的**关键** 。

#### **(5) 2024年12月 & 2025年11月（模拟/最新考点预测）**

* **横琴考察（2024.12）**：  
  *   
    **初心**：开发横琴的初心就是为了**澳门经济适度多元发展** 。  
  *   
    **四新产业**：科技研发和高端制造、中医药、文旅会展商贸、现代金融 。  
  *   
    **战略意义**：澳门+横琴是丰富“一国两制”实践的**新示范** 。  
* **梅州与广州考察（2025.11）**：  
  * **梅州（老区/山区）**：  
    *   
      **最大短板**：习近平指出，**城乡区域发展不平衡**是广东高质量发展的**最大短板** 。  
    *   
      **共同富裕**：小康路上“一个都不能少”，共同富裕路上\*\*“一个都不能掉队”\*\* 。  
    *   
      **南福村案例**：曾经“老少边穷”，现在种**金柚**致富（科技兴农、电商销售） 。  
  * **广州（创新）**：  
    *   
      **着力点**：在推动**科技创新和产业创新深度融合**上打头阵 。  
    *   
      **港珠澳大桥**：不仅是物理连接，更要实现人才流、资金流、信息流的畅通 。

## **第二部分：2025 广东高质量发展大会（省情专项）**

**核心逻辑**：高质量发展是首要任务 $\\rightarrow$ 根本途径是建设现代化产业体系 $\\rightarrow$ 关键变量是科技创新。

### **1\. 核心主题与战略**

*   
  **主题**：**建设现代化产业体系** 。  
*   
  **根本途径**：实现高质量发展的根本途径在于**建设现代化产业体系** 。  
*   
  **决定性变量**：抓住了**科技创新**，就抓住了现代化产业体系的**决定性变量** 。  
*   
  **主要抓手**：坚持**实体经济为本**、**制造业当家** 。

### **2\. 现代化产业体系的“四梁八柱”**

*   
  **构成**：现代化的**工业、农业、服务业**和**基础设施** 。  
*   
  **基本特征**：**智能化、绿色化、融合化** 。  
  * *记忆理解*：融合化指一二三产融合（如农业+文旅），也指科技与产业融合。  
*   
  **基本要求**：完整性、先进性、安全性 。

### **3\. 产业发展的具体举措**

* **“四要”口号**：  
  1. 要锻造先进制造**钢筋铁骨**。  
  2. 要点燃科技创新**澎湃引擎**。  
  3. 要育成现代企业**强壮主体**。  
  4. 要培厚发展环境**肥沃土壤** 。  
*   
  **两大重点领域**：尤其是抓住**人工智能**和**机器人**两大领域集中发力、抢占先机 。  
* **企业梯度培育（十百千万计划）**：  
  * **50家**科技领军企业。  
  * **100家**链主企业。  
  * **1000家**制造业单项冠军企业。  
  *   
    **10000家**专精特新企业 。

## **第三部分：广东省情与第十五届全运会（常识必考）**

### **1\. 广东省情概览**

* **地理特征**：  
  *   
    **“七山二水一分田”**：山地和丘陵面积约占**60%**，平原台地占40% 。  
  *   
    **海洋大省**：海域面积是陆地面积的**2.3倍**；大陆岸线长居**全国第一** 。  
  *   
    **地形地势**：北高南低；最高峰是**石坑崆**（海拔1902米） 。  
  *   
    **气候**：从北向南依次为中亚热带、南亚热带和热带气候 。  
* **经济地位**：  
  * 地区生产总值（GDP）连续**36年**居全国第一 。  
  * 区域创新能力连续**8年**全国第一 。  
  * 外贸进出口总额突破9万亿元 。  
*   
  **行政建制**：秦朝平定岭南后设南海、象郡、桂林三郡；\*\*“广州”**之名始于三国吴辖地；**“广东”\*\*得名于北宋初 。

### **2\. 第十五届全国运动会（2025年）**

*   
  **举办特色**：**粤港澳三地共同承办**（历史首次），香港、澳门首次承办全运会 。  
* **时间地点**：  
  * **开幕式**：2025年11月9日，**广州**（广东奥林匹克体育中心）。  
  *   
    **闭幕式**：2025年11月21日，**深圳**（宝安欢乐剧场） 。  
* **核心标识**：  
  *   
    **主题口号**：**“激情全运会，活力大湾区”** 。  
  *   
    **吉祥物**：**“喜洋洋”、“乐融融”** 。  
  *   
    **会徽**：图案由代表粤港澳三地的花瓣（**木棉、紫荆、莲花**）交叠旋转而成，立意为“繁荣、包容” 。  
  *   
    **火炬**：名为\*\*“绽放”**。核心理念为**“融”**（交流融合）与**“荣”**（共同繁荣）。燃料为**丙烷\*\* 。  
* **开幕式亮点**：  
  * 文体展演主题：**“圆梦未来”**。  
  * 点火仪式：**苏炳添、张家朗、李祎**（粤港澳三地运动员）共同点燃，火种源来自**深海可燃冰** 。

## **第四部分：习近平新时代中国特色社会主义思想（理论底座）**

### **1\. 核心体系框架**

*   
  **总论**：是马克思主义基本原理同中国具体实际相结合、同中华优秀传统文化相结合（**两个结合**）的重大成果 。  
*   
  **主要内容**：“十个明确”、“十四个坚持”、“十三个方面成就” 。  
* **世界观方法论（六个必须坚持）**：  
  1. 必须坚持**人民至上**（根本立场）。  
  2. 必须坚持**自信自立**（立足点）。  
  3. 必须坚持**守正创新**（原则）。  
  4. 必须坚持**问题导向**（路径）。  
  5. 必须坚持**系统观念**（基础方法）。  
  6. 必须坚持**胸怀天下**（格局） 。

### **2\. 中国式现代化（核心高频考点）**

* **五大特征**：  
  1. 人口规模巨大的现代化。  
  2. 全体人民**共同富裕**的现代化（本质特征）。  
  3. **物质文明和精神文明相协调**的现代化。  
  4. 人与自然**和谐共生**的现代化。  
  5. 走**和平发展道路**的现代化 。  
*   
  **重大意义**：打破了“现代化=西方化”的迷思，创造了**人类文明新形态** 。  
* **关键论断（易混淆点辨析）**：  
  *   
    **首要任务**：**高质量发展** 。  
  *   
    **本质要求**：发展**全过程人民民主** 。  
  *   
    **鲜明标识**：**开放** 。  
  *   
    **必然要求**：**城乡融合发展** 。  
  *   
    **重要保障**：**法治**；**高水平社会主义市场经济体制** 。  
  *   
    **基础性、战略性支撑**：**教育、科技、人才** 。

### **3\. 全面深化改革（二十届三中全会精神）**

*   
  **主题**：**进一步**全面深化改革、推进中国式现代化 。  
*   
  **总目标**：到2035年，全面建成**高水平社会主义市场经济体制** 。  
*   
  **七个聚焦**：聚焦构建高水平社会主义市场经济体制、聚焦发展全过程人民民主、聚焦建设社会主义文化强国、聚焦提高人民生活品质、聚焦建设美丽中国、聚焦建设更高水平平安中国、聚焦提高党的领导水平和长期执政能力 。  
*   
  **重大原则（六个坚持）**：坚持党的全面领导、坚持以人民为中心、坚持守正创新、坚持以制度建设为主线、坚持全面依法治国、坚持系统观念 。  
*   
  **牵引**：以**经济体制改革**为牵引 。

## **第五部分：重点领域专项考点（时政必刷）**

### **1\. “三农”与乡村振兴**

*   
  **战略总纲**：加快建设**农业强国** 。  
*   
  **重中之重**：解决好“三农”问题 。  
*   
  **底线任务**：巩固拓展脱贫攻坚成果、防止出现规模性返贫致贫 。  
*   
  **2025中央一号文件**：强调巩固和完善农村基本经营制度，坚持“大稳定、小调整”，**不允许城镇居民到农村购买农房、宅基地** 。

### **2\. 科技创新与数字中国**

* **新质生产力**：  
  * **核心标志**：全要素生产率大幅提升。  
  *   
    **特点**：创新。**关键**：质优。**本质**：先进生产力 。  
  *   
    **三化**：高科技、高效能、高质量 。  
* **数字中国**：  
  * “深入推进数字中国建设”是二十届四中全会明确任务 。  
  * 关键特征：数字化、网络化、智能化 。

### **3\. 国家安全与法治**

* **总体国家安全观**：  
  * **宗旨**：人民安全。  
  *   
    **根本**：政治安全（放在首要位置） 。  
  * **基础**：经济安全。  
* **法治建设**：  
  *   
    **全面依法治国**：是中国特色社会主义的本质要求和重要保障 。  
  *   
    **反分裂国家法20周年**：丰富完善“以法惩独”制度体系 。

### **4\. 党的建设**

*   
  **两个答案**：跳出历史周期率的第一个答案是“让人民来监督政府”，第二个答案是\*\*“自我革命”\*\* 。  
*   
  **根本政治立场**：**人民立场** 。  
*   
  **最大优势与危险**：最大政治优势是密切联系群众，执政后最大危险是脱离群众 。

## **备考策略总结（讲师经验提炼）**

1. **锁定“广东视角”**：复习时凡是看到\*\*“粤港澳大湾区”、“制造业当家”、“百千万工程”、“海洋强省”\*\*等字眼，必须高度敏感，这是广东卷的特色和命题重点。  
2. **区分“第一次”**：  
   * 十八大后总书记第一次视察地方是广东（2012）。  
   * 二十届四中全会后总书记第一次视察是海南和广东（2024/2025）。  
3. **精准记忆提法**：  
   * **根本保证** $\\rightarrow$ 党的领导。  
   * **根本途径/关键一招** $\\rightarrow$ 改革开放。  
   * **首要任务** $\\rightarrow$ 高质量发展。  
   * **本质要求** $\\rightarrow$ 共同富裕 / 全过程人民民主（视语境而定）。  
4. **关注数字**：如广东海岸线长度（3368.1公里）、全运会举办时间（11月9日）、2029年完成改革任务等，常作为单选题干扰项出现。
